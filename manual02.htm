[2] ГЛАВНОЕ ОКНО

Главное окно имеет три (или пять) кнопок в верхней части, и список съёмок внизу. Этот список пуст, пока вы не создадите съёмку. При первом запуске приложения отображается приветственное сообщение. Вы можете посмотреть его в любое время в меню Описание

Для выхода из главного окна TopoDroid Дважды нажмите на кнопку НАЗАД на вашем устройстве.

ДЕЙСТВИЯ

  • нажатие на съёмку: окно данных топосъёмки
  • Длинное нажатие на съёмку: окно свойств топосъёмки

КНОПКИ

  • Приборы: переключает в окно приборов
  • Новая съёмка
  • Импорт
    • топосъёмки или ZIP архива
  • Дополнительные кнопки [E] ThManager (менеджер Therion) и SQLite viewer; оба требуют стороннтх приложений

МЕНЮ

  • Рисование позволяет выбрать инструменты рисования для загрузки.
  • Совместная съёмка [E] открывает диалоговое окно, чтобы соединиться с TopoDroid приложением на другом Android устройстве для выполнения совместнй топосъёмки.
  • Настройки: все настройки программы.
  • Журнал настройки [E]. журнала в основном нужен для отладки приложения.
  • About: the program welcome message
  • Help shows a brief descriptions of buttons and menus. At the top-right corner there is a button that opens this user manual. Each TopoDroid window has a similar Help menu.

НАСТРОЙКИ

  • Текущий рабочий каталог. По умолчанию это "TopoDroid"
  • Язык
  • Размер текста элементов списков [по умолчанию 14, мин. 1]
  • Размер кнопок: "маленькие" для 5" экранов, "большие" для 7" планшетов [по умолчанию "маленькие"]
  • Клавиатура включает или отключает собственные клавиатуры TopoDroid .
  • Уровень функционала выбирает уровень функционала [по умолчанию "нормальный"].
  • Совместная съёмка [E] включить/отключить функцию совместной съёмки (очень экспериментальная).

Если блютуф не включен, приложение спросит, следует ли включить его. Без Bluetooth приложение может обмениваться данными с DistoX и данные съёмки могут быть введены только вручную. В этом случае кнопка Скачать в окне пикетов будет скрыта, и что в окне абрисов заменяется на Добавить.

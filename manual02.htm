[2] ГЛАВНОЕ ОКНО

Главное окно имеет три (или пять) кнопок в верхней части, и список съёмок. Этот список будет пуст, пока вы не создадите съёмку. При первом запуске приложения отображается приветственное сообщение. Вы можете посмотреть его в любое время в меню Описание

Для выхода из главного окна TopoDroid Дважды нажмите на кнопку НАЗАД на вашем устройстве.

ДЕЙСТВИЯ

  • нажатие на съёмку: окно данных топосъёмки
  • длинное нажатие на съёмку: окно свойств топосъёмки

КНОПКИ

  • Приборы: переключает в окно приборов
  • Новая съёмка
  • Импорт топосъёмки или ZIP архива
  • Дополнительные кнопки [E] ThManager (менеджер Therion) и SQLite viewer; оба требуют сторонних приложений

МЕНЮ

  • Палитра позволяет выбрать какие инструменты рисования загружать.
  • Совместная съёмка [E] открывает диалоговое окно для соединения с TopoDroid приложением на другом Android устройстве для выполнения совместнй топосъёмки.
  • Настройки: все настройки программы.
  • Журнал [E]. настройки журнала в основном нужен для отладки приложения.
  • Описание: сообщение приветствия с основными сведениями и указанием версии.
  • Помощь показывает краткое описание функций кнопок и меню. В правом верхнем углу диалога есть кнопка, которая открывает это руководство. Каждое окно TopoDroid имеет аналогичное меню Помощи.

НАСТРОЙКИ

  • Текущий рабочий каталог. По умолчанию это "TopoDroid"
  • Язык
  • Размер текста элементов списков [по умолчанию 14, мин. 1]
  • Размер кнопок: "маленькие" для 5" экранов, "большие" для 7" планшетов [по умолчанию "маленькие"]
  • Клавиатура включает или отключает собственные клавиатуры TopoDroid.
  • Уровень функционала выбирает уровень функционала [по умолчанию "нормальный"].
  • Совместная съёмка [E] включить/отключить функцию совместной съёмки (очень экспериментальная).

Если Bluetooth не включен, приложение спросит, следует ли включить его. Без Bluetooth приложение не может обмениваться данными с DistoX и данные съёмки могут быть введены только вручную. В этом случае кнопка Скачать в окне пикетов и в окне абрисов будет скрыта, она заменяется на кнопку Добавить.

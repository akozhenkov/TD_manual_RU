[2] ГЛАВНОЕ ОКНО

В главном окне есть четыре функциональные кнопки в левой верхней части экрана, кнопка выпадающего меню в правом верхнем углу и список съёмок под верхним рядом кнопок. Этот список будет пуст, пока вы не создадите съёмку. При первом запуске приложения отображается приветственное сообщение. Вы можете посмотреть его в любое время через пункт меню Описание.

Для выхода из главного окна TopoDroid дважды нажмите на физическую кнопку НАЗАД на вашем устройстве.

ДЕЙСТВИЯ

  • нажатие на съёмку: открывает окно данных топосъёмки
  • длинное нажатие на съёмку: открывает окно свойств топосъёмки

КНОПКИ

  • Приборы: переключает в окно устройств DistoX
  • Новая съёмка - открывает диалог создания новой топосъёмки
  • Импорт: топосъёмки или ZIP архива
  • Палитра: позволяет выбрать какие инструменты рисования загружать.
  • Менеджер: съёмок Therion. Позволяет соединять несколько съёмок при помощи приложения ThManager.

МЕНЮ

  • Палитра [A] открывает диалог для установки дополнительных наборов инструментов рисования
  • Журнал [E]: настройки журнала - в основном нужен для отладки приложения.
  • Описание: сообщение приветствия с основными сведениями и указанием версии.
  • Настройки: все настройки программы.
  • Помощь: показывает краткое описание функций кнопок и меню. В правом верхнем углу диалога есть кнопка, которая открывает это руководство (пока только на Английском языке). Каждое окно TopoDroid имеет аналогичное меню Помощи.

НАСТРОЙКИ

  • Рабочий каталог: По умолчанию это "TopoDroid"
  • Размер текста: элементов списков [по умолчанию 14, минимум 1]
  • Размер кнопок: маленькие, нормальные - для 5" экранов; средние, большие для 7" планшетов и огромные [по умолчанию - "маленькие"]
  • Уровень функционала: выбирает уровень функционала программы из: базовый, нормальный, продвинутый, экспертный. [по умолчанию "нормальный"].
  • Клавиатура: включает или отключает собственные клавиатуры TopoDroid.
  • Без курсора отключает или включает отображение курсоа в TopoDroid клавиатуры. [T]
  • Руководство на Руссом включает руководство пользователя на Руссом языке. Для активации этой функции нужно скачать переведенное руководство и поместить его в папку man в основном каталоге программы.

  • Язык
    • Из меню настроек "Главного окна" доступны все остальные диалоги настроек
    • Импорт/Экспорт
    • Данные съёмки
    • Абрисы
    • Устройства

    Если Bluetooth не включен, приложение спросит, следует ли включить его. Без Bluetooth приложение не может обмениваться данными с DistoX и данные съёмки могут быть введены только вручную. В этом случае кнопка Скачать в окне пикетов и в окне абрисов будет скрыта и заменена кнопкой Добавить.

[2] ГЛАВНОЕ ОКНО

В главном окне есть четыре кнопки в верхней части экрана, и список съёмок. Этот список будет пуст, пока вы не создадите съёмку. При первом запуске приложения отображается приветственное сообщение. Вы можете посмотреть его в любое время в меню Описание.

Для выхода из главного окна TopoDroid дважды нажмите на кнопку НАЗАД на вашем устройстве.

ДЕЙСТВИЯ

  • нажатие на съёмку: открывает окно данных топосъёмки
  • длинное нажатие на съёмку: открывает окно свойств топосъёмки

КНОПКИ

  • Приборы: переключает в окно приборов
  • Новая съёмка
  • Импорт: топосъёмки или ZIP архива
  • Дополнительные кнопки [E] ThManager (менеджер Therion) и SQLite viewer: оба требуют сторонних приложений
  • Палитра: позволяет выбрать какие инструменты рисования загружать.

МЕНЮ

  • Совместная съёмка [E]: открывает диалоговое окно для соединения с TopoDroid - приложением на другом Android устройстве для выполнения совместной топосъёмки.
  • Настройки: все настройки программы.
  • Журнал [E]: настройки журнала - в основном нужен для отладки приложения.
  • Описание: сообщение приветствия с основными сведениями и указанием версии.
  • Помощь: показывает краткое описание функций кнопок и меню. В правом верхнем углу диалога есть кнопка, которая открывает это руководство (пока только на Английском языке). Каждое окно TopoDroid имеет аналогичное меню Помощи.

НАСТРОЙКИ

  • Текущий рабочий каталог: По умолчанию это "TopoDroid"
  • Язык
  • Размер текста: элементов списков [по умолчанию 14, минимум 1]
  • Размер кнопок: "маленькие" для 5" экранов, "большие" для 7" планшетов [по умолчанию - "маленькие"]
  • Клавиатура: включает или отключает собственные клавиатуры TopoDroid.
  • Уровень функционала: выбирает уровень функционала [по умолчанию "нормальный"].
  • Совместная съёмка: [E] включить/отключить функцию совместной съёмки (очень экспериментальная).

Из меню настроек "Главного окна" доступны все остальные диалоги настроек

  • Импорт/Экспорт
  • Данные съёмки
  • Абрисы
  • Устройства

Если Bluetooth не включен, приложение спросит, следует ли включить его. Без Bluetooth приложение не может обмениваться данными с DistoX и данные съёмки могут быть введены только вручную. В этом случае кнопка Скачать в окне пикетов и в окне абрисов будет скрыта и заменена кнопкой Добавить.

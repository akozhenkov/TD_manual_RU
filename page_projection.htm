Projected profile azimuth

This dialog is opened when you create a new sketch with a projected profile.

The central canvas display the survey midline. It can be translated and zoomed.

The direction of view of the projected profile is set by dragging the seekbar at the top. The midline display rotates about the sketch reference station.

The "OK" button creates the sketch and opens it in the Sketch window.

The BACK button closes the dialog without creating the sketch.

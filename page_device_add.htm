Диалог добавления устройства

С помощью этого диалогового окна пользователь может добавить устройство в список устройств TopoDoid. В диалоге есть поля
  • Адрес
  • Модель и номер
  • Псевдоним (необязательно)

Адрес выбирается из выпадающего списка, в котором показаны MAC-адреса сопряженных устройств, которых нет в базе данных TopoDroid.
Адрес также можно узнать из реквизитов подключенного устройства в приложении «Настройки».

Модель выбирается из списка поддерживаемых устройств.

Номер должен быть четырехзначным. Оно может иметь ведущие нули. Номер устройства можно прочитать из информации на самом устройстве.

Псевдоним не является обязательным и его можно оставить пустым.

ПОИСК по данным калибровки

Этот диалог открывается при нажатии на кнопку Поиск в окне Данных калибровки.

Позволяет искать калибровочные данные с погрешностью, превышающей заданную, минимальную ошибку, которую можно ввести в текстовом поле.
Калибровочные данные со значением ошибки больше заданой минимальной ошибки выделено оранжевым фоном.

В диалоге существует три кнопки:

  • Закрыть диалог без каких-либо действий.
  • Отмена отображения результата поиска.
  • Поиск выполнить поиск и отобразить результат.

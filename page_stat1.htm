Survey statistics

This dialog is opened tapping the Statistics menu of the Survey info window.

The survey statistics are

  • number of (regular) legs and their total length
  • number of duplicate legs
  • number of surface leg
  • number of splay shots
  • number of stations
  • number of loops
  • number of disconnected surveys (this is 1 if the legs are all attached together in one survey)

Note: length are in the units set in the app settings.

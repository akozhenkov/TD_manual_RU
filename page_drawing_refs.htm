Настройки окна абрисов

Это диалоговое окно открывается с помощью кнопки Вид из окна Абрисов или окна Обзора.

  • Нитка: показывать нитку хода (белым цветом)
  • Бок. замеры: показывать боковые замеры (серым цветом)
  • Пикеты: показывать номера пикетов (фиолетовым)
  • Сетка:, показывать сетку
  • линейка, показать масштабную линейку
  • контуры, наложить контуры другого абриса
  • Шкала масштаба показать масштабную линейку
  • Контуры другого абриса показать другой абрис в виде тонких контуров
  • Сдвиг относительно нитки:, зафиксировать нитку хода и передвинуть рисунок
Существует три шага сетки,
  1. сетка с ячейкой 1 м (темно серая)
  2. сетка с ячейкой 10 м (серая)
  3. сетка с ячейкой 100 м (светло серая)

TopoDroid отображает только ту сетку, которая подходит для текущего масштаба абриса.

Если в съёмке есть еще абрисы, то можно наложить контур (серым цветом) другого абриса на текущий абрис чтобы легче было рисовать продолжение. При выборе этой опции откроется новый диалог, где можно выбрать абрис из списка либо очистить экран от выбранного абриса.

Последний пункт [Экспериментальный уровень] определяет, должна ли быть зафиксирована нитка при сдвиге абриса двумя пальцами. Это используется для совмещения абриса с ниткой после импорта абриса из Pocket Topo, в случае если импорт выполнен со сдвигом.

Если в съёмке сделано несколько абрисов, то можно отобразить еще один абрис в виде тонких серых контуров для того чтобы было легче продолжить рисование нового абриса. При выборе этой опции открывается диалог со списком абрисов в съемке, гдн можно выбрать нужный абрис для отображения или очистить предыдущий выбор.

Шестой пункт [Экспериментальный уровень] определяет, должна ли быть зафиксирована нитка при сдвиге абриса двумя пальцами. Это используется для совмещения абриса с ниткой после импорта абриса из Pocket Topo, в случае если импорт выполнен со сдвигом.

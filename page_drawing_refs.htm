Настройки окна абрисов

Это диалоговое окно открывается с помощью кнопки Вид из окна Абрисов или окна Обзора.

  • нитка, показывать нитку хода (белым цветом)
  • Бок. замеры показывать боковые замеры (серым цветом)
  • Пикеты names (violet)
  • Сетка, показывать сетку
  • Фиксир относительно(исправить) , зафиксировать нитку
Существует три шага сетки,
  1. сетка с ячейкой 1 м (темосерая)
  2. сетка с ячейкой 10 м (серая)
  3. сетка с ячейкой 100 м (светлосерая)

TopoDroid показывает только сетки, которые подходят для текущего масштаба абриса.

Пятый пункт [Эксперементальный уровень] определяет должна ли быть зафиксирована нитка при сдвиге абриса двумя пальцами. Это используется для совмещения абриса с ниткой после импорта абриса из Pocket Topo в случае импорт выполнен со сдвигом.

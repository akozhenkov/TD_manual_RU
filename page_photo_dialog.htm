ФОТО ПРИ ЗАМЕРЕ/НА АБРИСЕ

Этот диалог открывается при нажатии на кнопку Фото во вторичном диалоге Свойств замера, либо когда на абрисе ставится точка "фото", либо при создании фото поперечного сечения в окне абрисов.

В диалоге есть текстовое поле для коментария к фото которое вы собираетесть сделать.

Выключите опцию Камера TopoDroid если вы хотите использовать системное приложение камеры вместо камеры TopoDroid.

Кнопки:

  • OK закрывает диалог и делает снимок.
  • Отмена закрывает диалог.

Камера TopoDroid
Это упрощенное приложение камеры. Главное отличие этого приложения в том, что оно сохраняет азимут и наклон в exif свойствах файла снимка. Азимут записывается в поле GPS_LONGITUDE и наклон в поле GPS_LATITUDE (со знаком).
Следующие exif теги записываются TopoDroid:

  • Software: "TopoDroid" и его версия, разделенные пробелом
  • Orientation:
  • Datatime: время снимка
  • GPS latitude: наклон
  • GPS latitude ref: "N" "S"
  • GPS longitude: азимут
  • GPS longitude ref: "E" "W"
  • Image description: азимут и наклон в 1/100°, разделенные пробелом

Для точки "photo" или для фото сечения приложение камеры TopoDroid помещает перекрестье для удобства прицеливания.

В приложении камеры TopoDroid есть три кнопки внизу справа.

  • камера. Красный: готовность, нажмите чтобы сделать снимок. Голубой: ожидание, нажмите чтобы активировать камеру.
  • сохранить сохраняет снимок
  • назад закрывает диалог
Если экран телефона вверх то снимок будет портретным. Если телефон повернут на бок the снимок будет альбомным.

Кнопки зума появляются при касании нижней части экрана. Чтобы бысто менять зум жмите на кнопки быстрее.

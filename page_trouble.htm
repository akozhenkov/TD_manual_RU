DistoX решение проблем

поиск DistoX TopoDroid может не найти DistoX при первом сканировании, и вам, возможно, придется попробовать два или три раза, прежде чем DistoX будет найден.

Если обнаружить DistoX не удается несколько раз, вы можете попробовать сканирование через Настройки телефона. Если это приложение тоже не может найти DistoX, есть какието проблемы с Bluetooth на вашем Android и вы не сможете использовать TopoDroid с DistoX.

сопряжение DistoX Перед тем как TopoDorid сможет подключиться к DistoX, он должен быть сопряжен с Android. Если выбрана опция "Авто-сопряжение", диалоговое окно сопряжения должно появиться при выборе найденного DistoX после сканирования. Пин код DistoX 0000 (четыре нуля).

Также можно выполнить сопряжение Android с текущим DistoX через меню "сопряжения" в Настройках телефона.

После того, как DistoX, сопряжен с Android он будет сохранет и не будет запрашивать PIN-код еще раз. Если он продолжает запрашивать PIN-код каждый раз при загрузке данных из DistoX, попробуйте использовать "небезопасное соединение.

Если у вас есть проблемы с подключением и загрузкой данных попытатйтесь включить "решение для Z6".

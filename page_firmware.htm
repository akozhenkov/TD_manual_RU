DistoX firmware

TopoDroid includes the DistoX2 firmwares v. 2.1, 2.2, 2.3 and 2.4, which are all compatible with hardware 1.0. Through the firmware dialog you can save a copy of the current firmware of your DistoX2, or upload a new firmware file. Before uploading a new firmware TopoDroid checks that it is compatible with the hardware version. Unfortunately there is no sure way to read the firmware version from the file content, so TopoDroid relies on the signature in the file after the bootloader blocks. The DistoX2 must be booted in "bootloader mode". Refer to the DistoX2 documentation for details.
WARNING. UPLOADING AN INCOMPATIBLE FIRMWARE MAKES YOUR DistoX UNUSABLE.

Upon uploading a firmware, TopoDroid displays a message saying how many bytes have been written to the DistoX. If these are less than the firmware file size, the upload has failed.

Note
The firmware updates are always logged in the TopoDroid log file.

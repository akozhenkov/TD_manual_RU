ДИАЛОГ ПОИСКА ПИКЕТОВ

Этот диалог открывается при нажатии на кнопку "поиск" в окне данных съёмки.

В нем есть поле для ввода номера пикета, и чекбокс для включения сплеев в результаты поиска.

Кнопка поиск закрывает диалог и переводит спивок замеров к найденому пикету.
Есть также три кнопки для поиска замеров и одна для сплеев [T]:

  • дублирующие замеры
  • поверхностные замеры
  • замеры без направления развертки
  • обратные сплеи

Поиск производиится только в отображаемых данных и результаты поиска подсвечиваются желтым цветом в колонке номеров пикетов.
Долгое нажатие на кнопку поиск выводит следующий найденный пикет в верх списка.
Результат поиска сбрасывается при нажатии на любую кнопку.

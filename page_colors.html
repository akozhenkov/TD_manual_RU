Цветовая кодировка

Политика именования пикетов
  • forward leg [default]: синий
  • backward leg: тёмно-синий
  • foresight + backsight: фиолетовый
  • tripod: розовый
  • magnetic anomaly: красный
  • TopoRobot: оранжевый

Данные съемки
Цвет переднего текста:

  • белый: замер между пикетами
  • серый: один из повторяющихся замеров между пикетами
  • синий: сплей
  • зеленый: x-сплей
  • темно-синий: h-сплей
  • очень темно-синий: v-сплей
  • оранжевый: пустой замер
  • фиолетовый:
  • другие: назначенный пользователем цвет

Цвет фона строки:
  • чёрный: нормальный замер
  • красный: недостоверный замер
  • серый: закомментированный замер
  • оранжевый: короткий замер

Пикеты съёмки
Цвет текста:

  • зеленый: текущий пикет

Цвет фона:
  • черный: нормальный
  • тёмно-зелёный: последние измерения
  • серый: мульти выбор
  • темно-желтый: обратный замер в настройках DistoX

Замеры на абрисе

  • белый: нормальные
  • красный: недоставерные
  • оранжевый: несовместимость с братскими замерами

Сплеи

  • голубой: нормальные
  • темно-зеленый: закоментированные
  • зеленый, темно-синий, совсем темно-синий: классы сплеев
  • другие: назначенный пользователем цвет

Пикеты съёмки

  • фиолетовый: нормальные
  • зеленый: текущий пикет
  • красный: нитка хода скрытая после пикета
  • синий: нитка хода скрытая до пикета
  • желтый: сохраненный пикет

Данные калибровки
Цвет текста:

  • синий, оранжевый: группы замеров
  • серый: неиспользуемые данные

Цвет фона:
  • фиолетовый: не полностью скачанные данные (не используются)
  • красный: ошибка больше 1 градуса
  • зеленый: большое отклонение от среднего значения группы
  • коричневый: перенасыщенные данные
  • серый: удаленные данные (группа -1)

НАБОРЫ ИНСТРУМЕНТОВ РИСОВАНИЯ

В TopoDroid есть шесть наборов инструментов для рисования:
  • спелеология
  • археология
  • геология
  • биология
  • палеонтология
  • шахтёрское дело
По умолчанию установлен только набор спелеологических символов.
При помощи пункта "ПАЛИТРА" в меню основного окна можно вызвать диалог для установки дополнительных инструментов. Отметьте наборы инструментов, которые вы хотите добавить, и нажмите на кнопку "Установить".
При нажатии на кнопку "Заменить" уже установленные инструменты будут удалены перед установкой выбранных вами наборов.

После установки, необходимо включить нужные инструменты, воспользовавшись кнопкой "Палитра" в главном окне. Набор включеных инструментов рисования хранится в базе данных и остается неизменным, даже если некоторые инструменты были удалены.

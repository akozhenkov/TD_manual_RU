Поддиректории TopoDroid

На этой странице приводится некоторая информация о том, как хранятся данные съёмки, данные калибровки, а также абрисы. Вы можете смело пропустить этот и следующие два раздела, и вернуться к ним позже, но лучше все таки их прочитать.

Приложение создает свой базовый каталог, по умолчанию названный "TopoDroid" в основной внешней памяти устройства (обычно /SDCard). Этот каталог не удаляется при удалении приложения. В этом каталоге приложение создает две базы данных SQLite. Первая "device10.sqlite", где находятся данные калибровок, информация о приборах DistoX и другие данные. Вторая "distox14sqlite" она хранит данные топосъёмок. Приложение также создает подкаталоги для различных типов файлов:

  • bin: файлы прошивок DistoX2
  • csv: текстовые файлы CSV (экспортированные съёмки и калибровки)
  • cav: файлы Topo (экспортированные съёмки)
  • csx: файлы cSurvey (экспортированные съёмки и абрисы)
  • dat: файлы Compass (экспортированные съёмки)
  • dxf: файлы DXF (экспортированные съёмки и абрисы)
  • dump: дампы памяти DistoX
  • import: файлы съёмок для импорта (форматы Therion, Compass, VisualTopo, и PocketTopo)
  • kml: экспорт в GoogleEarth [A]
  • note: текстовые файлы заметок к съёмкам
  • photo: фото съёмок (в отдельную папку для каждой съёмки)
  • plt: экспорт треков в формате OziExplorer [A]
  • png: файлы изображений PNG (экспортированные абрисы)
  • srv: файлы Walls (экспортированные съёмки)
  • svg: файлы SVG (экспортированные абрисы)
  • svx: файлы Survex (экспортированные съёмки)
  • symbol: фалы инструментов рисования (точки, линии, области)
  • th: файлы Therion (сохраненные съёмки)
  • th2: файлы скрапов Therion (сохраненные абрисы)
  • th3: файлы 3D моделей
  • tmp: временные файлы. Любой файл, оставшийся в этом каталоге может быть безопасно удален.
  • top: файлы PocketTopo (экспортированные съёмки)
  • tro: файлы VisualTopo (экспортированные съёмки)
  • zip: съёмки в ZIP архивах (экспортированные и импортируемые)

Некоторые из этих подкаталогов используются только для хранения файлов экспорта (csx, dat, dxf, png, svx, th, top, tro). Другие используются для хранения данных (th2, th3, note, photo). Поддиректория "symbol" содержит файлы символов, используемые программой. Поддиректория "zip" используется для хранения архивов которые TopoDroid может читать и распаковывать. Когда вы хотите импортировать съёмку, сделанную в другой программе (Therion, Compass, VisualTopo, или PocketTopo) вы должны поместить файлы в подкаталог "import". Наконец "bin" используется для хранения файлов прошивки, как файлов прошивки, входящих в TopoDroid так и дампов прошивки из DistoX.

ПОДДИРЕКТОРИИ TopoDroid

На этой странице приводится некоторая информация о том, как хранятся данные съёмки, данные калибровки, а также абрисы. Вы можете смело пропустить этот и следующие два раздела, и вернуться к ним позже, но лучше все-таки их прочитать.

Приложение создает свой базовый каталог по умолчанию в основной внешней памяти, "Documents/TDX". Этот каталог не удаляется при удалении приложения. В этом каталоге приложение создает папку для съемок "TopoDroid" с базой данных SQLite "distox14.sqlite" в которой содержатся данные съемок, файлы абрисов и другие файлы съемок.

Приложение также создает подкаталоги для различных типов файлов.
Следующие подкаталоги создаются только в частном каталоге приложения "Android/data/com.topodroid.TDX/files" в основной внешней памяти:

  • bin: файлы прошивок для (DistoX v. 2 и BLE)
  • ccsv: текстовые файлы CSV (экспортированные калибровки)
  • dump: дампы памяти DistoX
  • man: первод руководства пользователя
  • symbol: фалы инструментов рисования (точки, линии, области)
В папке"bin" храняться файлы прошивок DIstoX, как оригинальные прошивки, так и дампы, скачанные с DistoX.
Файл "distox10sqlite" хранит данные калибровок и информацию о приборах, от также находится в этой директории.

Для каждой топосъемки создается папка-съёмки (названная также как называется съёмка) со следующими подкаталогами:

  • audio: аудио записи коментариев (wav files)
  • note: текстовый файл заметок к съёмке
  • photo: фото съёмок (jpg files)
  • tdr: абрисы во внутреннем формате
  • out: экспортироывнные файлы

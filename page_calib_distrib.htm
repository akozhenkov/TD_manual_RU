РАСПРЕДЕЛЕНИЕ ДАННЫХ КАЛИБРОВКИ (только для DistoX)

Это диалоговое окно открывается нажатием кнопки Распределение в окне Данных калибровки.

Оно отображает распределение данных калибровки, по азимуту и углу наклона, т.е. на 3D-сфере.

Сфера рассечена по горизонтальной плоскости, верхние и нижние полусферы отображаются в виде двух кругов. Центр кругов - это вертикальное направление, а точки на границе - горизонтальные направления. На каждом круге Север вверх, юг вниз, восток справа и запад слева.

Зеленые зоны соответствуют направлениям, хорошо покрытым данными. Красные зоны - это направления, для которых отсутствуют данные. В диалоге может отображаться либо угловое распределение сырых данных или данных, скорректированных с помощью калибровки.

Сфера раскрыта как элипс: север в середине, восток справа, запад слева и юг на внешней границе. Положительный наклон в верхней полусфере, отрицательный в нижней. Она может показывать либо угловое распределение исходных данных, либо данных, скорректированных калибровкой.

КНОПКИ
Кнопка Сырые вычисляет угловое распределение с использованием исходных значений направления.
Кнопка С учетом калиб. вычисляет распределение после коррекции данных калибровочными коэффициентами.

G отображает распределение векторов ускорения в данных DistoX
M отображает распределение векторов магнитного поля в данных DistoX

Разворот отображает распределение углов разворота данных DistoX

Качественный набор калибровочных данных не должен иметь красных пятен в распределениях.

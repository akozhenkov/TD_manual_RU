Это диалоговое окно открывается при нажатии на кнопку Память в окне Приборов.

Дамп памяти

Кнопка Дамп считывает содержимое памяти DistoX. Чтение памяти DistoX представляет собой медленный процесс, и для этого DistoX должен быть "включен". Желательно считать небольшую часть памяти, скажем, 50 записей единовременно. Необходимо ввести начальный и конечный адреса памяти в двух текстовых поля справа. Если вы ввели имя файла, он также сохраняется в файл дампа (в папке "dump").

Просмотр памяти требует некоторого знания о внутренностях DistoX: как минимум, адресного пространства и способа хранения данных. Вы должны прочитать документацию DistoX перед использованием.

Содержимое памяти DistoX отображается в виде списка в диалоговом окне "Список памяти". Каждая строка содержит:

  • адрес памяти
  • буквенный код который, обозначает относится ли запись к нормальному измерению ('d') или калибровочному измерению ('g')
  • значения памяти.

Обычные замеры показаны как длинна, (в метрах), азимут и угол (в градусах).
Показания "точности" замеров в DistoX2 не показываются.
Для данных калибровки показаны сырые данные (значения 'm' не показаны для DistoX2).
Записи неизвестного типа отмечены вопросительным знаком ('?').

Только для DistoX A3:

  • Прочитать указатели памяти
  • Сохранить указатели памяти
  • Cбросить часть памяти, т.е., пометить ее "для скачивания"

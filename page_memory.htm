DistoX memory

This dialog is opened tapping the Memory button of the Device window.

  • Dump reads the content of the DistoX memory. You must enter the initial and final memory location in the two text fields to its right.
  • Optional filename to save the dump

Only for DistoX A3:
  • Read the memory cursors
  • Store the memory cursors
  • Reset a portion of memory, ie, marks it "to-be-downloaded"
The content of the DistoX memory is displayed in the "Memory list" dialog.

Reading the DistoX memory directly is a slow process, and the DistoX must be kept "ON" while doing it. It is advisable to read a portion of memory, say 50 entries, at a time. Be patient: this operation is carried out in the main gui thread.

If you have entered a filename it is also saved to a dump file (in the "dump" folder).

Memory inspection requires some knowledge of the internals of the DistoX: at a minimum the address space, and the way data are stored. You should have read the DistoX documentation before using it.

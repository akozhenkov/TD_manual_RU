3D VIEWER - Поверхность DEM

Модель поверхности DEM (Digital Elevation Model) может быть добавлена в 3D модель Topodroid или показана, если она встроена в файл модели (например в lox файл), или был добавлена в 3D модель.

Кнопка Поверхность окна 3D-просмотра включает и выключает отображение грида высот.

Этот диалог открывается из меню Поверхность окна 3D-просмотра.

Он используется для загрузки DEM, текстуры поверхности для наложения на DEM и настройки ее непрозрачности.
Он содержит

  • кнопка для выбора файла DEM
  • кнопка выбора файла текстуры, плсле того как в модель загружен файл DEM
  • ползунок для регулировки непрозрачности
  • флажок для включения отображения проекции модели пещеры на поверхность
  • галочка для включения отображения текстуры поверхности (если есть)

DEM файл
Средство трехмерного просмотра поддерживает файлы DEM в формате ascii или в формате грида Therion (see manual for details on the file formats and coordinate reference systems).

Файл текстуры
Средство трехмерного просмотра поддерживает файлы изображений текстур GeoTIFF и файлы карт OSM.

Координаты X-Y файла DEM и текстуры должны быть в системе координат модели пещеры (which can be changed in the fixed station dialog), поскольку программа 3D-просмотра не выполняет преобразование координат.

Файлы OSM находятся в географических координатах WGS84. Значения можно использовать с моделями, в проекции UTM, но преобразование приблизительное.

Замер на абрисе

Этот диалог откроется, если выбрать вектор и нажать на кнопку Примечания в окне Абриса (в режиме "поточечного" редактирования).

Свойства замера:

  • Длина, азимут и угол
  • пикеты
  • примечание
  • разрез развертка: "влево"(-1), "вертикально"(0) или "вправо"(1)
  • разрез частичная развертка: допонительная развертка в пределах [-0.5, 0.5]
  • флаг дубликат
  • флаг поверхность
  • флаг примечание
  • опции для того чтобы скрыть часть нитки хода

Кнопка Сохр подтверждает изменения и закрывает диалог.

Кнопки Назад и Отмена закрывают диалог без сохранения изменений.

Скрыть нитку хода
Замеры между пикетами имеют опцию сксрыть нитку хода до или после одного из пикетов. Скрывающие пикеты выделены на абрисе красным или фиолетовым цветом. Нитку можно показать изменив соответствующие опции в диалоге свойств пикета.

Направление пазверки замера [T]
Замеры на разрезе-развертке обычно направлены влево, вверх, или впрево (также морут вообще не отображаться на развертке). Для выбора направления задается значение параметра "развертка", значение определяет направление развертки: лево (-1), вверх (0), право (+1). При выборе замера между пикетами на абрисе разреза-развертки появится горизонтальная полоса, проходящая через центр замера. Проводя курсором вдоль полосы в одну или другую строну от центра полосы пользователь может быстро сменить "направление развертки" замера.

Частичная развертка [T]
Также возможна "частичная развертка", значения между -0.5 и 0.5. Эти значения можно выбрать при помощи слайдера "развертка" над опциями развертки. If not modified the fractional extend of a shot is zero.

Частичная развертка будет равна нулю если:

  • the shot extend is set with a multishot action (except for the "flip" action)
  • the shot extend is reset in the drawing window using the yellow extend line
  • the shot extend is changed in the shot edit dialog (of the shot window)
  • the shot extend is assigned by TopoDroid

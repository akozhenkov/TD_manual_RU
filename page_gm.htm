Данные калибровки

Это диалоговое окно открывается нажатием на строку данных в окне Данных калибровки.

  • азимут, угол, разворот вокруг оси
  • ошибка: калибровки
  • группа: номер группы
  • перегруппировать: переприсвоить номера групп начиная с текущей строки
  • удалить: данные
  • Сохранить: изменения

Каждой строке может быть присвоен номер группы.

  • положительный: замер принимает участие в вычислении калибровочных коэффициентов
  • ноль: замер не используется в вычислении калибровочных коэффициентов
  • -1 (отрицательный): "удаленные" данные. Это может быть сделано также с помощью кнопки Удалить
Строки с одинаковым положительным номером группы образуют группу.

Удаленные данные не удаляются из базы.
Они обычно не отображаются в списке данных калибровки, не принимаются во внимание при автоматическом назначении групп, и не вносят вклад в вычисление коэффициента.

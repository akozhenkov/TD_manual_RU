Calibration data

This dialog is opened tapping a data entry in the Calibration data window.

  • azimuth, inclination, roll
  • calibration error
  • group number entry field
  • regroup: reassign the groups from this data onward
  • delete the data
  • Save the changes

The item group can be set or changed.

  • positive: the data takes part to the computation of the calibration coefficients
  • zero: the data does not contribute to the computation
  • -1 (negative one): "delete" the data. This can be done also with the Delete button
Data with the same positive group number form a group.

Deleted data are not removed the data from the database.
They are not usually shown in the calibration data list, are not taken into account in the automatic group assignment, and do not contrinute to the coefficient computation.

[5] ДАННЫЕ КАЛИБРОВКИ

Окно данных калибровки отображает список калибровочных замеров. Открывается по нажатию на кнопку "открыть" в окне калибровки. Если в калибровке еще нет данных, в окне не будет ничего, кроме кнопок. Каждая строка данных калибровки содержит следующую информацию:
  • индекс: порядковый номер, который идентифицирует измерение
  • группа: номер группы, к которой принадлежат измерения
  • азимут, угол наклона и угол разворота DistoX вокруг своей оси, вычисленные из исходных значений без коррекции калибровки
  • ошибка: разница между скорректированными калибровкой направлениями и осредненным по группе направлением, в градусах

Калибровочные замеры отображаются красным и синим цветом, чередуясь между четными и нечетными номерами групп. Каждая группа представляет собой набор измерений, сделанных в одном направлении, которые отличаются только углом поворота DistoX вокруг своей оси. Данные, не имеющие группы, отображаются серым цветом.

ДЕЙСТВИЯ

  • нажатие на строку данных: открывает диалог редактирования калибровочных измерений

КНОПКИ

  • Скачать: загрузить данные с DistoX
  • Переключить режим: переключает DistoX в режим калибровки и обратно
  • Bluetooth: сброс соединения BT или удаленное управление DistoX
  • Группирование: автоматическая разбивка калибровочных замеров на группы
  • Рассчитать калибровку: вычисляет калибровочные коэффициенты и показывает результат
  • Распределение: отображает угловое распределение калибровочных замеров
  • Считать: выгружает и отображает калибровочные коэффициенты, хранящиеся в DistoX
  • Записать: передает калибровочные коэффициенты в DistoX. Доступно только после того, как калибровочные коэффициенты были вычислены. Коэффициенты могут быть записаны напрямую через диалог, отображающий результат вычисления.

На данный момент активна только функция загрузки калибровочных замеров по требованию. Загруженные данные добавляются к текущей таблице калибровочных замеров. Вы можете сделать несколько калибровочных измерений, загрузить их и проверить. Если вы не удовлетворены их качеством, можно продолжить измерения, пока вы не будете удовлетворены вашими данными калибровки. Загруженные данные могут быть "отключены" (путем установки номера группы на "0" (ноль) - (отключенные данные не используются при расчете калибровочных коэффициентов), или удалены (как правило, не отображаются в списке данных). Удаленные данные не учитываются при группировании.

Bluetooth

Эта кнопка сбрасывает соединение Bluetooth в TopoDroid.
Он также позволяет удаленно управлять DistoX [A]: включение или выключение лазера, выполнение калибровочного измерения. Данные немедленно загружаются, если выбрана соответствующая настройка.

Расчет калибровки

Запустит расчет калибровочных коэфициентов можно нажав на кнопку с шестеренкой.
После того, как расчет закончен, будет показан диалог со значениями коэффициентов.
В случае если исходных данных было недостаточно или могим замерам не был присвоен номер группы, то калибровка не может быть расчитана и будет показано предупреждающее сообщение.
Коэффициенты не могут быть расчитаны если достигнуто максимальное число итераций и при этом не получено решение. В этом случае будет показано соответствующее сообщение.

Внимание

  • Сообщение - "насыщенные" данные: означает, что было обнаружено магнитное поле с аномально высокими значениями. Возможной причиной этого является наличие паразитных магнитных полей. Насыщенные данные отображаются на красном фоне.
  • Сообщение - нулевые данные: данные калибровки были не корректно переданы из DistoX. Будет показана строка данных с нулями. По умолчанию ей не будет присвоен номер группы, но при этом она будет учитываться при расчете номеров группы для остальных строк. Нулевые данные отображаются на розовом фоне.

МЕНЮ

  • Показать: нужно ли отображать все данные, в том числе удаленные
  • Валидация: проверка этой калибровки при помощи другой калибровки
  • Настройки: те же настройки, что доступны из окна калибровки

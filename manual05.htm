[5] ДАННЫЕ КАЛИБРОВКИ

Окно данных калибровки отображает список калибровочных замеров. Если в калибровке еще нет данных, в окне не будет ничего кроме кнопок. Каждая строка данных калибровки содержит следующую информацию:
  • индекс: порядковый номер, который идентифицирует измерение
  • группа: номер группы к которой принадлежат измерения
  • азимут, угол наклона и угол разворота вокруг оси, вычисленные из исходных значений без коррекции калибровки
  • ошибка: разница между скорректированными калибровкой направлениями и осредненным по группе направлением, в градусах

Калибровочные замеры отображаются с красным и синим цветом чередуясь между четными и нечетными номерами групп. Данные, не имеющие группы отображаются серым цветом.

ДЕЙСТВИЯ

  • нажатие на строку данных: открывает диалог редактирования калибровочных измерений

КНОПКИ

  • Скачать: получть данные с DistoX
  • Переключть режим переключает DistoX в режим калибровки и обратно
  • Группирование автоматическая разбивка калибровочных замеров на группы
  • Вычисление вычисляет калибровочные коэффициенты и показывает результат
  • Распределение отображает угловое распределение калибровочных замеров
  • Считать отображает коэффициенты, хранящиеся в DistoX
  • Записать передает калибровочные коэффициенты в DistoX. Доступно, только после того, как калибровочные коэффициенты были вычислены. Коэффициенты могут быть записны напрямую через диалог отображающий результат вычисления.

На данный момент активна только функция загрузки калибровочных замеров по требованию. Загруженные данные добавляются к текущей таблице калибровочных замеров. Вы можете сделать несколько калибровочных измерений, загрузить их и проверить, если вы не удовлетворены их качеством можно продолжить измерения пока вы не будете удовлетворены вашими данными калибровки. Загруженные данные могут быть "отключены" (путем установки номера группы на "0" (ноль); (отключеные данные не используются при расчете калибровочных коэффициентов), или удалены (как правило, не отображаются в списке данных). Удаленные данные не учитываются при группировании.

Внимание. Сообщение о том, что есть "насыщенные" данные означает, что было обнаружено магнитное поле с аномально высокими значениями. Возможной причиной этого является наличие паразитных магнитных полей. Насыщенные данные отображаються на красном фоне.

МЕНЮ

  • Показать: нужно ли отображать все данные, в том числе удаленных данные
  • Валидация: проверка этой калибровки при помощи другой калибровки
  • Настройки: те же, что и для окна калибровки

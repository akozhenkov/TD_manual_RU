Sensors

This dialog is opened tapping the Sensor button in the secondary Shot edit dialog.

It has list a checkboxes to select a builtin sensor, or an external sensor (in this case you must enter the "sensor type").

There are three text fields

  • type of sensor. Either one of the listed builtin sensors, or typed if if you select the "external" sensor
  • values Automatically updated for builtin sensors
  • comment
Save stores the measurement in the database.

The Back key closes the dialog without saving.

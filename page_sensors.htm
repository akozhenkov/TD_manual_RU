Датчики

Это диалоговое окно открывается нажатием кнопки Датчики во вторичном диалоге Редактирования измерений.

В нем есть несколько опций по выбору встроенных датчиков, или внешних датчиков (в этом случае вам нужно ввести "тип измерения").

Доступно три текстовых поля:

  • тип датчика. Либо один из встроенных датчиков, либо тот, который вы впишете туда при выборе "внешнего" датчика
  • показания: водятся автоматически для встроенных датчиков
  • комментарий
Кнопка Сохранить записывает измерение в базу данных.

Кнопка Назад закрывает диалог без сохранения.

Датчики

Это диалоговое окно открывается нажатием кнопки Датчики во втором диалоге Редактирования измерений.

В нем есть несколько опций по выбору встроенных датчиков, или внешних датчиков (в этом случае вам нужно ввести "тип измерения").

Доступно три текстовых поля

  • тип датчика. Либо один из встроенных датчиков, либо то чтов напишите туда при выборе "внешнего" датчика
  • показания Водятся автоматически для встроенных датчиков
  • комментарий
Сохранить записывает измерение в базу данных.

Кнопка Назад закрывает диалог без сохранеия.

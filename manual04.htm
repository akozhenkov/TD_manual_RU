[4] ОКНО КАЛИБРОВКИ

При нажатии Калибровка в меню Устройств в окне будут показаны списки калибровок текущего DistoX, сдесь вы можете создать новую калибровку или открыть существующую. В обоих случаях вы попадаете в окно Калибровки.

КНОПКИ

  • Сохранить калибровочную информацию в базу данных
  • Открыть окно данных калибровки
  • Показать калибровочные коэффициенты. Это показывает ранее вычисленые коэффициенты, которые хранятся в базе данных

МЕНЮ

  • Экспорткалибровки
  • Удалить калибровку и ее данные из базы данных (необратимое действие)
  • Настройки
  • Справка

НАСТРОЙКИ КАЛИБРОВКИ

  • Правила группирования: как разбивать калибровочные замеры на группы [по умолчанию "группировать по 4"]
  • Допуск в группе: максимальное угловое расстояние между последовательными замерами для того чтобы они попали в одну группу в случае если используется "основанная на расстоянии" политика группирования [по умолчанию 40, минимум 0]
  • Ошибка: вычисление коэффициентов заканчивается, если изменение каждого коэффициента во время следующей итерации меньше этого значения [по умолчанию 10-6i, минимум 0].
  • Итерации: максимальное число итераций алгоритма калибровки [по умолчанию 200, минимум 50].
  • Исходные данные: следует ли отображать необработанные данные. По умолчанию TopoDroid отображает азимут, наклон и разворот вокруг оси. Отметьте эту опцию, если вы хотите, чтобы увидеть также необработанные данные калибровки. [по умолчанию "нет"]
  • Алгоритм: может быть 0 (авто), 1 (линейный), 2 (нелинейный). Если алгоритм калибровки остается "авто", при этом параметр в настройках не "авто", то алгоритм из настроек используется для расчета калибровки. Если оба параметра на"авто" алгоритм выбирается автоматически в зависимости от модели DistoX и версии прошивки. [по умолчанию "авто"]

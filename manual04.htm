[4] ОКНО КАЛИБРОВКИ

При нажатии на кнопку Калибровка (шетерёнка) в окне Устройств будет показан список калибровок для текущего DistoX.
Здесь вы можете создать новую калибровку или открыть существующую. В обоих случаях откроется в окно Калибровки. Также есть возможность импортировать калибровку из CSV файла, находяещгося в папке ccsv.

Окно калибровки содержит текстовые поля для:

  • Название. Каждая калибровка длжна иметь уникальное название
  • Дата
  • Описание
А также опции выбора для алгоритма расчета калибровки. По умолчанию установлено в "авто" (тоесть, TopoDroid сам выбирает подходящий алгоритм).

КНОПКИ

  • Сохранить: калибровочную информацию в базу данных
  • Открыть: окно данных калибровки
  • Показать: калибровочные коэффициенты. То есть, показать ранее вычисленные коэффициенты, которые хранятся в базе данных

МЕНЮ

  • Экспорт: калибровки в файл CSV
  • Удалить: калибровку и ее данные из базы данных (необратимое действие)
  • Настройки
  • Помощь

НАСТРОЙКИ КАЛИБРОВКИ

  • Группирование данных калибровки: как разбивать калибровочные замеры на группы [по умолчанию "группировать по 4"]
  • Допуск в группе: максимальное угловое расстояние между последовательными замерами для того, чтобы они попали в одну группу в случае, если используется "основанная на расстоянии" политика группирования [по умолчанию 40, минимум 0]
  • Ошибка: вычисление коэффициентов заканчивается, если изменение каждого коэффициента во время следующей итерации меньше этого значения [по умолчанию 10e -6, минимум 0].
  • Итерации: максимальное число итераций расчета калибровки [по умолчанию 200, минимум 50].
  • Удаленное управление немедленное скачивание измерений.
  • Сырые данные: следует ли отображать необработанные данные. По умолчанию TopoDroid отображает азимут, наклон и разворот вокруг оси. Отметьте эту опцию, если вы хотите видеть сырые данные калибровки (компоненты G и M) в десятичном или шестнадцатиричном формате. [по умолчанию "нет"]
  • Алгоритм: может быть 0 (авто), 1 (линейный), 2 (нелинейный). Если алгоритм калибровки остается "авто", но при этом параметр в настройках не "авто", то для расчета калибровки используется алгоритм из настроек. Если оба параметра на"авто", алгоритм выбирается автоматически в зависимости от модели DistoX и версии прошивки. [по умолчанию "авто"]

DistoX Справочная информация

Краткое описание функций DistoX.
Официальное полное руководство доступно на странице http://paperless.bheeh.ch

Коды ошибок:

  • 252: слишком высокая температура
  • 253: слишком низкая температура
  • 255: слишком слабый сигнал
  • 256: слишком сильный сигнал
  • 257: слишком высокая освещенность
  • 260: препятсвия на пути лазерного луча

  • DIST: ВКЛ / включить лазер / измерение
  • CLR: отменить текущую операцию, выключить лазер
  • CLR (2 секунды): выключить прибор
  • MEM: данные, хранящиеся в памяти
  • REF: переключить точку отсчета

DistoX2 - X310

  • TIMER (лазер включен): запустить таймер (автоматическое измерение)
  • TIMER (лазер выключен): установить время таймера (при помощи PLUS/MINUS)
  • SMART: доп. информация об измерении
  • FUNC: информация о приборе

Долгое нажатие 2с:

  • DIST (лазер включен): измерение
  • CLR: выключение
  • CLR+SMART: переключить в режим калибровки
  • CLR+MEM: очистить память
  • CLR+FUNC: вкл/выкл блютуф
  • CLR+MINUS: запретить выключение
  • REF: точка осчета передний край
  • REF+FUNC: точка отсчета задний край
  • REF+PLUS: вкл/выкл подсветки
  • REF+MINUS: вкл/выкл звуковой сигнал
  • MEM: единицы измерений длинны
  • MEM+SMART: единицы измерений угла (deg/grad)
  • MEM+FUNC: включить silent mode
  • MEM+MINUS: переключть в режим обратных замеров [2.4]
  • SMART+MINUS: проверка троекратным измерением
  • DIST (with laser on): measure
  • CLR: power off
  • CLR+SMART: toggle calib. mode
  • CLR+MEM: clear unsent memory
  • CLR+FUNC: toggle bluetooth
  • CLR+MINUS: locked power off
  • REF: front reference
  • REF+FUNC: back reference
  • REF+PLUS: toggle display light
  • REF+MINUS: toggle beep
  • MEM: distance units
  • MEM+SMART: angle units (deg/grad)
  • MEM+FUNC: toggle silent mode
  • MEM+MINUS: toggle backsight mode [2.4]
  • SMART+MINUS: toggle triple-shot check [2.4]

Долгое нажатие 5с:

  • FUNC+SMART: тип источника питания
  • FUNC+CLR+MEM: сброс всех настроек
  • DIST+PLUS+MINUS (switching on): режим загрузчика
  • FUNC+SMART: change battery type
  • FUNC+CLR+MEM: factory reset
  • DIST+PLUS+MINUS (switching on): bootloader mode

доп информация о измерении:

  1. azimuth, H distance, V distance, distance
  2. roll, dip
  3. M, G
  4. on-board accel. raw x,y,z (*)
  5. separate accel. raw x,y,z (*)
  6. magnetic raw x,y,z (*)

  1. combined accel. x,y,z
  2. magnetic x,y,z
  3. on-board accel. x,y,z (*)
  4. separate accel. x,y,z (*)

Device info:

  1. battery
  2. hw/fw version, serial number
  3. display light level (1:10) [2.3]
  4. endpiece offset (-128:127) [2.3]

DistoX - A3

  • AREA (back ref.): continuous display of azimuth+clino (*)
  • AREA+REF (front ref.): continuous display of roll+dip (*)
  • AREA*2: firmware version and serial number
  • UNITS: toggle display light
  • AREA*2+REF*2+CLR: toggle silent mode
(*) Do not take measurements in this mode

Two seconds:

  • UNITS (in back ref.): distance units
  • UNITS (in front ref.): cycle angle units / BT
  • MEM*10+UNITS: toggle calib. mode

Five seconds:
  • CLR+PLUS: toggle beep

"Angle units / BT" cycle
  • compass off, BT off
  • compass on (grad), BT off
  • compass on (grad), BT on
  • compass on (deg), BT off
  • compass on (deg), BT on

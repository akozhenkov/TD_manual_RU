БАЗЫ ДАННЫХ

Полезно иметь общее представление об организации информации и данных в базах данных TopoDroid. Рабочими единицами в TopoDroid являются съёмки и калибровки. Съёмка представляет собой одну непрерывную топосъёмку участка пещеры. Калибровка является одним набором данных и калибровочных коэффициентов DistoX.
Съёмки сохраняются в базу данных "distox14.sqlite". Может существовать только одна такая база данных в каждой рабочей директории.
Калибровки сохраняются в базе данных "device10.sqlite", вместе с информаций о DistoX. Эта база данных является глобальной и присутствует только в каталоге "TopoDroid" .

Соответственно, наиболее важная таблица в базе данных:

  • surveys: информация о топосъёмках

Данные о съёмках содержат название съёмки, дату, команду, магнитное склонение, номер начального пикета и, возможно, комментарий.
Связанными с таблицей surveys являются таблицы для:
  • shots: замеры угла азимута и расстояния
  • stations: сохраненные пикеты
  • plots: 2D абрисы; планы, разрезы, поперечные сечения
  • photos: фотографии, ассоциированные с пикетами
  • audios: аудиозаписи коментариев, асоциированные, с замерами
  • sensors: показания датчиков проассоциированные с пикетами
  • fixeds: пикеты с заданными географическими координатами
  • sketches: 3D модели

Наиболее важные таблицы в базе данных устройств:

  • devices: информация о приборах
  • calibs: калибровки

Записи calibs содержат название калибровки, дату, устройство DistoX, алгоритм и результаты последнего вычисления: коэффициенты, среднюю ошибку, максимальную ошибку, число итераций.

Связанной с таблицец калибровок calibs является таблица с данными калибровки:

  • gms: калибровочные измерения

Наконец каждая база данных имеет таблицу:

  • configs: настройки

Вы можете узнать больше об организации данных в базе, открыв ее в SQLite просмотрщике.

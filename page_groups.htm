Группирование данных калибровки

Это диалоговое окно открывается нажатием кнопку Присвоить номер группы калибровочным замерам в окне Данных калибровки.

Перед вычислением коэффициентов калибровки калибровочные измерения должны быть разделены на группы данных, имеющих одинаковый азимут и угол, но разный разворот вокруг оси.

Эти группы пронумерованы целыми числами, начиная с 1.
Данные с номером группы 0 отображаются в списке, но исключены из расчета коэффициентов.
"Удаленные" данные имеют номер группы равный -1.

Можно назначить номер группы для каждой строки данных калибровки вручную (в диалоге "Редактировать данные калибровки" ), либо TopoDroid может назначать номера групп автоматически. Существуют три принципа автоматической группировки:

  1. по четыре замера в группе;
  2. по четыре замера для первых 16 измерений (четыре группы), после чего каждый замер в отдельной группе;
  3. группирование, основанное на относительном угловом расстоянии (разница в направлении) между замерами.

Действия:

  • Сбросить группы: сбрасывает все номера групп на 0
  • OK: назначает номера групп в соответствии с выбранным правилом (делает это для строк без номера группы)

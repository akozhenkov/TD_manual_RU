ГРУППИРОВАНИЕ ДАННЫХ КАЛИБРОВКИ

Это диалоговое окно открывается нажатием кнопку Присвоить номер группы калибровочным замерам в окне Данных калибровки.

Перед вычислением коэффициентов калибровки калибровочные измерения должны быть разделены на группы данных, имеющих одинаковый азимут и угол, но разный разворот вокруг оси.

Эти группы пронумерованы целыми числами, начиная с 1.
Данные с номером группы 0 отображаются в списке, но исключены из расчета коэффициентов.

Можно назначить номер группы для каждой строки данных калибровки вручную (в диалоге "Редактировать данные калибровки" ), либо TopoDroid может назначать номера групп автоматически. Существуют два принципа автоматической группировки:

  1. TopoDroid: все измерения по четыре замера в каждой группе;
  2. PocketTopo: по четыре замера для первых 16 измерений (четыре группы), после чего каждый замер в отдельной группе;

Действия:

  • Сбросить группы: сбрасывает все номера групп на 0
  • OK: назначает номера групп в соответствии с выбранным правилом (делает это для строк без номера группы и не помеченых как удаленые)

Группы данных калибровки

Это диалоговое окно открывается по нажатию на кнопку Сгруппировать в окне Данных калибровки.

Перед вычислением коэффициентов калибровки калибровочные измерения должны быть разделены на группы данных, имеющих одинаковый азимут и угол но разный разворот вокруг оси.

Эти группы пронумерованы целыми числами, начиная с 1.
Данные с номером группы 0 отображаються в списке, но исключены из расчета коэффициентов.
"Удаленные" данные имеют номер группы равный -1.

Можно назначить номер группы для каждой строки данных калибровки вручную (в диалоге "Редактировать данные калибровки" ), либо TopoDroid может назначать номера групп автоматически. Существуют три принцыпа автоматической группировки:

  1. по четыре замера в группе;
  2. по четыре замера для первых 16 измерений (четыре группы), после чего каждый замер в отдельной группе;
  3. группирование основанное на относительном угловом расстоянии (разница в направлении).

Если выбрана опция Сбросить группы номера групп будут переназначены с начала списка. Если эта опция не выбрана, то номера групп назначаються только записям, у которых еще нет номера.

OK закрывает диалог и назначает номера групп.

Режим отображения данных топосъёмки

Этот диалог открывается по нажатию на кнопку Вид в окне Списка пикетов.

  • ID пикетов идентификатор записи
  • боковые замеры: вектора, у которых, существует только пикет ОТ
  • пустые: замеры, в которых, не указаны пикеты
  • повторные измерения: повторные замеры между пикетами, которые, почти совпадают друг с другом

Идентификаторы записей, это уникальные идентификаторы, индексированные по возрастанию, присвоенные TopoDroid каждому замеру в съёмке.

Если боковые замеры скрыты то их можно показать для каждого пикета отдельно. При нажатии на номер пикета в строке измерения, будут показаны боковые замеры для этого пикета. Повторное нажатие на номер пикета (даже в списке боковых замеров) скрывает их.

Режим отображения данных топосъёмки

Этот диалог открывается по нажатию на кнопку Вид в окне Списка пикетов.

  • идентификатор записи
  • боковые замеры: вектора у которых существует только пикет ОТ
  • пустые: замеры в которых не назначены пикеты
  • повторный замер: замеры которые почти совпадают с предыдущими

Идентификаторы записей, это уникальные идентификаторы индексированные по возрастанию, присвоенные TopoDroid каждому замеру в съёмке.

Если боковые замеры скрыты то их можно показать для каждого пикета отдельно. При нажатии на название пикета в строке измерения будут покааны боковые замеры для этого пикета. Повторное нажатие на имя пикета (даже в списке боковых замеров) скрывает их.

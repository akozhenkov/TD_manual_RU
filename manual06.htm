[6] ОКНО ДАННЫХ ТОПОСЪЁМКИ

Окно Данных топосъёмки отображает список пикетов в съёмке.

Заголовок окна отображает название съёмки, а в режиме мульти дисто - активный DistoX. Цвет заголовка зависит от текущей политики нумерации пикетов:

  • голубой: прямые замеры с пикета ОТ
  • темно синий: обратные замеры с пикета ДО
  • розовый: режим "штатив"
  • фиолетовый: режим "backsight"
  • красный: "магнитная аномалия" policy
  • оранжевый: режим "toporobot"

TopoDroid различает четыре вида измерений (показаны разными цветами):

  1. [красный] "пустые" измерения: измерения, где номера пикетов не были установлены.
  2. [голубой] "боковой замер": сплей или луч или свободный замер, у него есть только один пикет "ОТ" и пустое поле "ДО" Эти замеры разделены на категории: X-splay (зеленый) или Гор./Верт. для плана и разреза (темно синий)
  3. [темно голубой] гор/верт-сплеи
  4. [зеленый] замеры ЛПВН перпендикулярны к нитке
  5. [белый] "нормальный" замер: у этого измерения есть как пикет "ОТ", так и пикет "ДО"
  6. [желтый]: "обратный" замер (при использовании схемы с обратным замером)
  7. [серый] "повторный замер" (следует сразу за нормальным): такому замеру не обязательно устанавливать пикеты
  8. [оранжевый] пустые замеры, т.е, замеры которым пока еще не были присвоены номера пикетов
  9. [фиолетовый]: пустой замер

Цвета фона строк:

  • светло-голубой (пикеты): последние загруженные измерения.
  • красноватый (измерения): ненадежные измерения, то есть, измерения с "аномальной" напряженностью магнитного поля (только Disto X2).
  • оранжевый (измерения): слишком маленькое расстояние между пикетами.
  • серый: закомментированный (временно исключенный) замер

ДЕЙСТВИЯ

  • нажатие на данные измерений в строке замера: открывает диалог редактирования измерений
  • нажатие на пикет: позволяет показать/скрыть боковые замеры с этого пикета, в случае если показ боковых замеров глобально отключен
  • длинное нажатие на пикет: позволяет выбрать/отменить выбор "текущего" пикета
  • длинное нажатие нажатие на длину или азимут бокового замера открывает последний абрис, отцентрованный на пикете, с которго сделан боковой замер. При этом боковой замер подсвечен красным цветом.

КНОПКИ

  • Bluetooth
  • Режим отображения данных в таблице
  • Список абрисов.
  • Заметки относящиеся к топосъёмке
  • Добавить измерение вручную
  • Сохраненные пикеты
  • <Направление разреза развертки
  • Поиск пикетов/замеров
  • Обновить список пикетов

Долгое нажатие на:

  • "Скачать двнные с DistoX": переключение DistoX (только в режиме мульти)
  • "Абрис": открывает последний абрис
  • "Данные введенные вручную": диалог поправок к измерениям
  • "Поиск пикетов": переход к следующему измерению

Одновременный выбор нескольких замеров:

  • установить развертку всех замеров "влево"
  • развернуть направление развертки
  • установить развертку всех замеров "вправо"
  • задать цвет для выбранных боковых замеров
  • перенумеровать замеры (the first must be a leg)
  • расчитать плоскость заложения замеров
  • удалить все выбранные замеры. Внимание: скрытые повторные измерения, соответствующие выбранным замерам тоже будут удалены
  • закрыть функцию множественного выбора

Диалог абрисов

С помощью диалога абрисов вы можете создать новый абрис или открыть существующий. В диалоге есть кнопка для создания нового абриса и список уже существующих в данной съёмке абрисов из двух колонок (в левой колонке абрисы плана, в правой абрисы разреза-развертки: открыв любой из них, вы можете переключиться на другой в окне рисования).

Если установлены компоненты для отображения 3D, также появится кнопка для создания новой 3D-модели, а существующие 3D модели будут перечислены после абрисов.

МЕНЮ

  • Свойства съёмки: информация о топосъёмке
  • Закрыть
  • Восстановить: восстановление удаленного элемента (пикет или абрис)
  • Фото: фотоснимки сделанные в этой съёмке
  • Датчики: список показаний датчиков в этой съёмке
  • 3D: 3D модель съёмки (требуется программа Cave3D)
  • DistoX: переключает на окно приборов
  • Настройки
  • Помощь

НАСТРОЙКИ СЪЁМКИ

  • Группа съемки: группа съемки по умолчанию. Если значение заполнено, то оно используется, чтобы установить поле "Группа" при создании новой топосъёмки. Впишите свое имя и других спелеологов, с которыми обычно делаете съемки. [По умолчанию пусто]
  • Настройки нумерации пикетов: выберите способ автоматической нумерации пикетов. [По умолчанию "ЛПВН+потом на след. пикет"]
  • Номера пикетов: определяет тип наименования пикетов - либо буквенно-цифровой либо цифровой [По умолчанию "буквенно-цифровой"]
  • Начальный пикет: номер по умолчанию для первого пикета в новой топосъёмке. Может быть изменен в диалоге Новой топосъёмки [По умолчанию не задан]
  • Размер миниатюры: Размер мини-копии фотографий [По умолчанию 200, минимум 80, максимум 400]
  • Авто-экспорт: Автоматически экспортировать данные съемки в формат по умолчанию при каждом закрытии окна данных [По умолчанию нет]
  • Фиксированныя точка отсчета абрисов: использовать одну точку отсчета координат для всех абрисов [По умолчанию нет]
  • Общие сечения: общие поперечные сечения на пикете [По умолчанию нет]
  • ЕДИНИЦЫ ИЗМЕРЕНИЯ
    • Единицы измерения длины: в метрах или футах [по умолчанию метры]
    • Единицы измерения углов: градусы или грады [по умолчанию градусы]
    • Единицы измерения шага сетки в абрисах: метры, ярды, 2 фута или 0,1 метр [по умолчанию метры]
  • НАСТРОЙКИ ДАННЫХ ПИКЕТОВ
    • Допустимое расхождение повторных измерений: угловой допуск (в процентах) между последовательными измерениями в отрезке [по умолчанию 0.05, минимум 0]. Допуск по длине является произведением допуска по углу и длины.
    • Максимальная длина замера: максимальное расстояние между пикетами, которое не расценивается как "слишком далеко", замеры помечаются как "удаленные" [по умолчанию 50 м, мин 20 м]
    • Минимальная длинна замера: Замеры расстояний между пикетами, меньшие чем заданный порог длинны будут выделены оранжевым цветом. [по умолчанию 0.5 м
    • Мин. кол-во измерений между пикетами: минимальное количество близких последовательных измерений для одного отрезка между пикетами.[по умолчанию 2, минимум 2, максимум 4]
    • Режим обратного замера DistoX: Если DistoX работает в режиме обратных измерений [по умолчанию выкл.]
    • Порог вертикальности бокового замера определяет как делать "развертку" бокового замера [по умолчанию 10, минимум 0, максимум 90]
    • Порог вертикальности для отрезков (Therion "vthreshold"): ЛПВН для измерений вводимых вручную преобразуется в Запад-Восток-Север-Юг в случае если вертикальный угол на следующий пикет превышает заданный порог [по умолчанию 80, минимум 0, максимум 90]
    • Устанавливать "развертку" вручную: либо фиксированная (лево/право) либо относительно азимута направления развертки [по умолчанию относительно азимута]
    • Закрывать петли автоматически и распределять ошибки. Желательно не делать компенсацию, так как это позволяет увидеть величину невязки и определить возможную ошибку в съёмке [по умолчанию выкл.] Распределение невязок в кольце основано на "треугольниках" - трилатерация [E].
    • Кнопки Пред/След. Показывать кнопки Предыдущий/Следующий пикет в диалоге измерений. По умолчанию - ВКЛ
    • Обратный замер: поле для ввода обратного замера в диалоге ручного ввода данных [N] [по умолчанию выкл.]
    • Таймер задержки измерения: кол-во секунд задержки перед измерением направления при помощи датчиков Android [по умолчанию 10, минимум 0]
    • Громкость: сигнала таймера [N] [по умолчанию 50, минимум 10, максимум 100]
  • МЕСТОПОЛОЖЕНИЕ настройки
    • Формат координат. {ггммсс, доли градусов}. По умолчанию ггммсс
    • CRS система координат. Введите название системы координат как показано в Proj4. По умолчанию Long-Lat
  • ТОЧНОСТЬ настройки
  • Максимальное допустимое ускорение. Максимальное ускорение при выполнении измерения, порог > 0. По умолчанию 1
  • Максимальная допустимая намагниченность. Максимальная напряженность магнитного поля при измерении, порог > 0. По умолчанию 1
  • Максимально допустимый магнитный угол. Максимально допустимое отклонение угда магнитного поля при замере, угловой порог > 0. По умолчанию 3 градуса

Спец настройки: Частичная развертка, Последние измерения и вермя их "свежести", Классы сплеев и их цвета, Подводная съёмка, Сенсоры.

Для выхода из окна данных топосъёмки дважды нажмите на кнопку НАЗАД на вашем телефоне.
Если настройка Авто-экспорта активирована и устаеновлен формат по умолчанию, то данные съёмки будут автоматически экспортироваться при каждом закрытии окна данных топосъёмки.

[6] ОКНО ДАННЫХ ТОПОСЪЁМКИ

Окно данных топосъёмки отображает список пикетов в съёмке.

TopoDroid различает четыре вида измерений:

  1. "пустые" измерения, измерения где пикеты еще не были установлены
  2. "боковой замер": или луч у него есть только один пикет, как правило, пикет "От"
  3. "нормальный" замер: у этого измерения есть как пикет "ОТ" так и пикет "ДО"
  4. "повторный замер" (следует сразу за нормальным): такому замеру не обязательно устанавливать пикеты

Каждый вид замеров отображается с другим цветом: "нормальный" белым, "повторный замер" серым, "боковой замер" голубым, "пустой" красным. Кроме того, голубоватый цвет фона отображается для последних загруженных измерений, красноватый для ненадежных измерений, то есть. измерения с "аномальной" напряженностью магнитного поля (только Disto X2).

ДЕЙСВИЯ

  • нажатие на строку: диалог редактирования измерения
  • длинное нажатие на строку: диалог редактирования дополнительных параметров измерения
  • нажатие на пикет: показать/скрыть боковые замеры с этого пикета
  • длинное нажатие на пикет: выбрать/отменить выбор "текущего" пикета

КНОПКИ

  • DistoX скачать данные. Скрыта, если не выбран ни один DistoX
  • Bluetooth
  • Режим отображения данных в таблице
  • Список абрисов. Длинное нажатие кнопки сразу открывает самый последний абрис
  • Заметки относящиеся к топосъёмке
  • Направление развертки
  • Сохраненные пикеты
  • Добавить измерение вручную

Диалог абрисов

С помощью диалога абрисов вы можете создаеть новый абрис или открыть существующий. В диалоге есть кнопка для создания нового абриса и список уже существующих в данной съёмке абрисов из дух колонок (в левой колонке абрисы плана, в правой абрисы разреза-развертки, открыв любой из них вы можете переключиться на другой в окне рисования).

Если установленны компоненты для отображения 3D, также появится кнопка для создания новой 3D-модели, а существующие 3D модели перечислены после абрисов.

МЕНЮ

  • Метаданные: информация о топосъёмке
  • Восстановить: восстановление удаленного элемента (пикет или абрис)
  • Фото: фотоснимки сделанные в этой съёмке
  • Датчики: показантя датчиков в этой съёмке
  • 3D: 3D модель съёмки (требуется Cave3D)
  • DistoX: окно приборов
  • Настройки
  • Справка

НАСТРОЙКИ СЪЁМКИ

  • Группа съемки: группа съемки по умолчанию. Если значение заполнено, то оно используется, чтобы установить поле "Группа" при создании новой топосъёмки. Напишите свое имя и других спелеологов с которыми обычно делаете съемки. [По умолчанию пусто]
  • Настройки нумерации пикетов: выберите способ автоматической нумерации пикетов. [По умолчанию "ЛПВН+потом на след. пикет"]
  • Номера пикетов определяет тип названия пикетов, либо буквенно-цифровой либо цифровой [По умолчанию "буквенно-цифровой"]
  • Начальный пикет: имя по умолчанию для первого пикета в новой топосъёмке. Может быть изменен в диалоге Новой топосъёмки [По умолчанию не задан]
  • Размер миниатюры: Размер мини-копии фотографий [По умолчанию 200, минмум 80, максимум 400]
  • ЕДИНИЦЫ ИЗМЕРЕНИЯ >>
    • Единицы измерения длинны: либо в метрах или футах [по умолчанию метры]
    • Единицы измерения углов: либо в градусах или грады [по умолчанию градусы]
    • Единицы измерения в абрисах: либо в метрах или в ярдах [по умолчанию метры]
  • НАСТРОЙКИ ДАННЫХ ПИКЕТОВ >>
    • Допустимое отклонение в измерении отрезков: угловой допуск (в процентах) между последовательными имерениями в отрезке [по умолчанию 0.05, минмум 0]
    • Мин. кол-во измерений в отрезке(пикеты отрезка): минимальное количество близких последовательных измерений для одного отрезка между пикетами.[по умолчанию 2, минмум 2, максимум 4]
    • Вертикальный порог бокового замера определяет как делать "развертку" бокового замера [по умолчанию 10, минмум 0, максимум 90]
    • Вертикальный порог для отрезков (Therion "vthreshold"): ЛПВН для измерений вводимых вручную преобразуется в Запад-Восток-Север-Юг в случае если вертикальный угол на след. пикет превышает заданный порог [по умолчанию 80, минмум 0, максимум 90]
    • Развертка отрезков: либо фиксированная (лево/право) либо относительно азимута направления развертки [по умолчанию выкл.]
    • Закрывать петли автоматически и распределять ошибки. Желательно не делать компенсацию, так как это позволяет увидеть величину невязки и определить возможную ошибку в съёмке [по умолчанию выкл.]
    • Обратный замер поле для ввода обратного замера в диалоге ручного ввода данных [N] [по умолчанию выкл.]
    • Магнитная аномалия: компенсация локальной магнитной аномалии. Она требует политики автоматического обратного визирования пикета [по умолчанию выкл.]
    • Таймер: кол-во секунд задержки перед измерением направления при помощи датчиков Android [по умолчанию 10, минмум 0]
    • Громкость: сигнала таймера [N] [по умолчанию 50, минмум 10, максимум 100]

  • ТОЧНОСТЬ настройки

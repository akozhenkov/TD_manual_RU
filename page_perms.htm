TopoDroid РАЗРЕШЕНИЯ

Хранилище данных
TopoDroid сохраняет данные съёмки и абрисы, а также калибровки во внешнем хранилище телефона.
Он не может работать без доступа на чтение / запись к хранилищу.

Internet
Подключение к сети интернет необходимо только в том случае, если вы хотите установить перевод руководства пользователя.
TopoDroid не использует сеть любым другим способом, и это разрешение можно отключить.

Bluetooth
TopoDroid требуется доступ по Bluetooth для связи с прибором.
Если данные съёмки вводятся вручную, Topodroid можно использовать без Bluetooth.
TopoDroid также использует Bluetooth-сопряжение для недавно обнаруженных устройств. Обнаружение и сопряжение можно выполнить с помощью приложения «Настройки телефона».

Камера
TopoDroid может использовать камеру для съемки или перенаправить задачу в стандартное приложение «камера».
Управление камерой TopoDroid не так богато, как в приложении «камера». TopoDroid сохраняет направление взгляда (азимут и наклон) в снимках.
Вы можете безопасно использовать TopoDroid без этого разрешения.

Аудио
TopoDroid использует микрофон для записи звуковых заметок (для замеров и аудио точек).
Если это разрешение не предоставлено, аудиозапись отключена.
Вы можете безопасно использовать TopoDroid без этого разрешения.

Местоположение
TopoDroid использует устройство GPS для определения географического положения точки.
Вы можете безопасно использовать TopoDroid без этого разрешения.

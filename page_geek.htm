Спец настройки

Спец настройки разных продвинутых фич TopoDroid.
Они доступны только на теством уровне функционала.
The flags marked with "*" are disables by default.

  • Частичная развертка нитки хода [*]
  • Последние измерения выделены синим цветом и отображаются выборочно [*]
  • Vertical threshold for legs [80]
  • Splay dash coherence: splay shots dashed coherently between plan and profile views [enabled]
  • Порог угла для боковых замеров пунктиром: боковые замеры с углами больше этого порогового значения будут отображаться на плане пунктиром. [50 градусов]
  • Порог азимута для боковых замеров пунктиром: боковые замеры с разницей в азимуте к замерам больше этого порогового значения будут отображаться на разрезе пунктиром. [60 deg.]
  • Порог угла для боковых замеров пунктиром: боковые замеры с углами меньше этого порогового значения, будут отображаться на поперечном сечении пунктиром [60 deg.]
  • Number of sketch backups, kept in a round list
  • Minimum interval between two backups [60 s]

Line settings

  • Minimal corner angle in making lines straight (fourth button of line-item edit dialog) [45]
  • Interpolation accuracy for cubic Bezier segments [1.0]
  • Corner threshold for cubic Bezier segments [20.0]
  • Line/Area snap actions [*]
  • Smooth/straighten segment actions [*]
  • Multipath actions [*]

Auto wall settings

  • Auto-wall tyoe (none, convex, Delaynay) [none]
  • Maximum clino for plan-view splays [70]
  • Minimum clino for profile-view splays [45]
  • Maximum separation of wall points along the shot [0.1]
  • Maximum concavity [0.1]
  • Distance between points in the wall-line [1.0]

Настройки устройсва DistoX

  • Задержка подключения: задержка [1/10 сек], пауза между попытками соединения с DistoX [по умолчанию - 0, отсутствие задержки]
  • Пауза скачивания: пауза после загрузки одного измерения [мс] [по умолчанию 250]
  • Пауза ожидания готовности: пауза ожидания готовности данных [мс] [по умолчанию 500]
  • Пауза вкл. лазера: пауза после включения лазера [мс] [по умолчанию 1000]
  • Пауза после замера: пауза после выполнения измерения [мс] [по умолчанию 2000]

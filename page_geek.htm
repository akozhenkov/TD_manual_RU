СПЕЦ НАСТРОЙКИ

Спец настройки разных продвинутых фич TopoDroid.
Они доступны только на теством уровне функционала.
Тем не менее, эффекты этих настроек могут быть видны на более низком уровне функционала (обозначены в квадратных скобках).
Функции по умолчанию отключены.

Общие настройки

  • Дополнительные наборы условных обозначений: установить инструменты из других наборов, в дополнение к основному набору инструментов для пещер
  • Журнал пакетов комуникации Дисто: пакеты сырых данных копируются в упакованную базу данных
  • Редактированиае абрисов: включает инструмент редактирования абрисов th2 Главное окно

Настройки данных топосъёмки

  • Подводная съёмка включает режим вода данных "подводной съёмки" - расстояние, азимут и глубина.
  • Недавние замеры выделить синим цветом и отображать селективно
  • Время в течении которого замеры считаются свежими
  • Неполная развертки нитки хода
  • Менять местами пикеты если включена опция обратный замер DistoX
  • Плоскостная интерполяция сплеев в общую поверхность и вычисление азимута и угла поверхности (элементы залегания пласта, для геологов)
  • Датчики использовать показания датчиков телефона [A]
  • Закрытие колец: метод компенсации невязки закрытия колец [E]
  • Порог невязки колец: порог невязки при селективном закрытии колец [E]
  • Using Android compass/clino [A]
  • Закрываь кольца Компенсация ошибок измерений при закрытии колец [E]
  • Использовать компас телефона для измерения азимута/наклона [по умолчанию откл] [A]
  • Таймер задержки измерений при удаленном управлении Дисто [E]
  • Громкость сигнала таймера измерений [E]
  • Промахи в серии измерений между пикетами [T]
  • Перенумерация пикета у сплеев
  • Перенумерация группы сплеев

Настройки данных сплеев

  • Классы сплеев: разделение сплеев на классы (гор. верт. сечение и обычные)
  • Цвета сплеев: включить возможность вручную назначать сплеям определенный цвет
  • Порог угла сплеев в плане (вертикальности) для замеров [80°] от горизонта [A]. Сплеи с углом превышающим значение порога не будут отображаться в плане.
  • Порог угла для сплеев пунктиром: сплеи с углами больше этого порогового значения будут отображаться на плане пунктиром. [50°]
  • Отображение сплеев пунктиром: сплеи отображаются обычной линией, либо пунктиром на плане и разрезе по одинаковым критериям, либо в зависимости от проекции (план, разрез) [вкл]
  • Порог азимута для боковых замеров пунктиром: боковые замеры с разницей в азимуте к замерам больше этого порогового значения будут отображаться на разрезе пунктиром. [60°]
  • Порог угла для боковых замеров пунктиром: боковые замеры с углами больше этого порогового значения, будут отображаться на поперечном сечении пунктиром [60°]
  • Цвета сплеев пунктиром
  • Цвета сплеев точкой
  • Цвета последних сплеев

Настройки рисования

  • Сдвиг и скалирование абриса относительно нитки
  • Разделить/склеить абрис
  • Размер стилуса [пиксели, 0: игнорировать]: отфильтровывать прикосновения пальцами при рисовании стилусом. [T]
  • Кол-во бекапов абриса для функции отмены действия [по умолчанию 5] [A]
  • Минимальный интервал между бекапами [60сек] [A]
  • Выделить сохраненные пикеты (реперы) оранжевым цветом на абрисах
  • Периодичность обновления нитки Пересчитывать и перерисовывать нитку хода только при скачивании новых измерений.(в режиме постоянной передачи данных) [not T]
  • Афинное преобразование включить афинное преобразование абриса.
  • Включть использование слоев абриса и типов объектов [по умолчанию нет]
  • Масштаб миллиметровки коррекция [0]
  • Наклонные сечения: разрешить ли наклонный поперечный разрез для разреза [нет]

Обновление только замеров не работает в режиме передачи данных по-требованию. Оно также не происходит в режиме фунционала тестера, поскольку, в этом режиме используется экспериментальное инкрементальное обновление.

Если наклонные сечения включены, плоскость сечений на разрезах может иметь наклон. Используются только наклоны, кратные 10. Если наклонные сечения не включены, линии разреза нарисованные более чем заданый порог будут вертикальными, а остальные горизонтальными.

Настройки линий

  • Минимальный угол спрямления линий (четвертая кнопка диалога редактирования линий) [45]
  • Точность интерполяции кривых безье. По умолчанию - 1
  • Угол Безье. Угловой порог кубической интерполяции кривых Безье. По умолчанию 20
  • Максимальное расстояние до точки/отрезка (Буфер прореживания) при прореживании [0.5 m]
  • Wedding segment buffer: buffer around a line segment [10 pxl]
  • Максимальная длина сегмента при прореживании [2.0 m]
  • Привязать Линии/Области к другим линиям, сплеям вкл/выкл функций [#]
  • Сгладить/спрямить сегмент вкл/выкл функций [#]
  • Спрямить линию, спрямить всю линию целиком.
  • Выбор нескольких линий для совершения действий с ними [#]
  • Комбинированные дейсвия

Настройки устройсва съёмки

  • Задержка подключения: задержка [1/10 сек], пауза между попытками соединения с прибором [по умолчанию - 0, отсутствие задержки] [B]
  • Второй прибр: позволяет топодройду принимать данные с двух приборов (DistoX only)
  • Пауза скачивания: пауза после загрузки одного измерения [мс] [по умолчанию 250] [A]
  • Пауза ожидания готовности: пауза ожидания готовности данных [мс] [по умолчанию 500] [A]
  • Пауза вкл. лазера: пауза после включения лазера [мс] [по умолчанию 1000] [A]
  • Пауза после замера: пауза после выполнения измерения [мс] [по умолчанию 2000] [A]
  • Прошивка проверка на совместимость [да] [A]
  • Режим BRIC4: прием базоваго или полного набора данных. С полным набором данных TopoDroid использует в качестве индекса замера, индекс передаваемый BRIC4. [all] [T]

Настройки импорта-экспорта

  • Включить архив условных обозначаний в экспортируемый zip файл
  • Импортируемый формат съемки при импорте (обычный, подводный) [normal]
  • Экспорт csx в формат cSurvey-transfer (необходим для cSurvey версии 1.20 и новее)
  • Точки пикетов: автоматически экспортировать точки пикетов в th2
  • ЛПВН кол-во сплеев: количество сплеев используемое для ЛПВН

Изменение данных пикетов (2)

Этот диалог открывается по длинному нажатию на строку в Таблице пикетов.

Диалог отображает номера пикетов и данные измерений (длинну, азимут, угол).

При уровнях функционала "Базовый" или "Нормальный" в этом диалоге будет одна кнпка Удалить измерение.

При более высоких уровнях функционала в диалоге будет пять кнопок:

  • диалог Фото
  • диалог Показаний датчиков
  • Добавить замер перед текущим (введя данные измерений вручную)
  • Разделить съёмку на этом пикете (создается новая съёмка)
  • Удалить замер

Кнопка удалить не удаляет замер из базы данных. Замер помечается как "удаленный" и может быть восстановлен позже через диалог "Восстановления".

Разделить съёмку
Эта кнопка открывает диалоговое окно для ввода информации о новой топосъёмке. Посде чего данные съёмки будут перенесены в новую топосъёмку, начиная с этого пикета. Затем программа продолжает в новой съёмке. Возможно, вам придется разделить съёмки, когда у вас есть данные из двух (отдельных) частей пещеры, хранящихся в DistoX: они загружаются в одну съёмку, которая должна быть затем разделена соответствующим образом.

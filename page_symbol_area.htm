ИНСТРУМЕНТ "ОБЛАСТЬ"

Инструменты "области" заданы в текстовых файлах.

В программу встроены два инструмента "область", "пользовательский" и "вода". Остальные загружаются из подкаталога TopoDroid area. Этот каталог заполняется инструментами "областей", упакованными в дистрибутив APK. Вы можете удалить инструменты, которые вам не нужны, редактировать те, которые вы хотите изменить, и добавлять новые, не вошедшие в APK.

Файл инструмента "область" содержит данные, определяющие свойства инструмента рисования:

  • "symbol area" [обязательный]
  • "name NAME", где NAME название области по умолчанию [обязательный]
  • переводы названия, т.е. "name-es ..." [не обязалельный]
  • "th_name NAME", где NAME это название в Therion, и должно совпадать с именем файла [обязательный]
  • "color rgb alpha" rgb и alpha - цвет в шестнадцатиричном формате например '0x0000ff' голубой, и '0x99' полупрозрачный [обязательный]
  • "close-horizontal" замыкать горизонтально [не обязательный]
  • "orientable" ориентируемый [не обязательный]
  • "level" view_flag [не обязательный, 1 if missing]
  • "bitmap WIDTH HEIGHT X_TILE Y_TILE" - начало растрового рисунка. Режимы заливки могут быть: M (зеркальный) или R (повторение). За этой строчкой должны следовать: HEIGHT (высота) линий, WIDTH (ширина) и пиксели рисунка (1: рисунок, 0: фон).
  • "endbitmap" отмечает конец растра
  • "roundtrip VALUE" [не обязательный для SVG-roundtrip, VALUE может быть 1 (walls_shp), 2 (walls_sym), 3 (detail_shp), 4 (detail_sym) - по умолчанию 3]
  • "endsymbol"
Название
Правила для названий символов описаны в Point Tool.

Замыкать горизонтально
Если этот атрибут установлен, то полигон замыкается горизонтально на разрезах и в сечениях.
Если Y координата последней точки нарисованной линии близка к Y координате первой точки, то область автоматически замыкается горизонтальной линиией. Таким образом вы можете рисовать только нижнюю границу области на разрезе.
Если Y координата последней точки нарисованной лини сильно отлчается от первой, область будет закрыта наклонным сегментом.

В настоящее время этот атрибут используется только для встроенной области "вода".

Растровый рисунок
С помощью этой опции можно задать фоновый узор, например, для дифференциации пещерных образований.

В настоящее время этот атрибут не используется ни в одном инструменте, поставляемом в APK.

Ориентация
Если установлен атрибут "ориентируемый", то заливку области можно вращать с помощью диалога редактирования области.
Заливки областей не используются в наборе символов по умолчанию. Заливка цветом используется для повышения производительности.

Слой
Атрибут "level" считывает значение флага для того чтобы отображать область только на определенном слое.
Флаг это сумма чисел 1 (базовый), 2 (пол), 4 (отложения), 8 (потолок) и 16 (артефакты).

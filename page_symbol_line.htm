ИНСТРУМЕНТ ЛИНИИ

Инструменты "линий" заданы в текстовых файлах.

Инструменты линий, "пользовательский" и "стены", встроены в программу. Остальные загружаются из подкаталога TopoDroid symbol/line. Этот каталог заполняется инструментами линий, упакованных в дистрибутив APK. Вы можете удалить инструменты, которые вам не нужны, редактировать те, которые вы хотите изменить, и добавлять новые, не вошедшие в APK.

Файл инструмента линий содержит данные, определяющие свойства инструмента рисования.

  • "symbol line" [обязалельный]
  • "name NAME", где NAME название линии по умолчанию [обязалельный]
  • переводы названия, т.е. "name-es ..." [не обязалельный]
  • "th_name NAME", где NAME это название в Therion, и должно совпадать с именем файла [обязалельный]
  • "group GROUP", определяет к какой группе принадлежит линия [не обязалельный, по умолчанию ни к какой]
  • "color rgb alpha" (rgb и alpha цвет в шестнадцатиричном формате) [обязалельный, альфа не обязалельно]
  • "width SIZE" [не обязалельный, множитель "базовой" толщины линии]
  • "dash PATTERN" [не обязалельный рисунок штриховки]
  • "style STYLE" где STYLE может быть 'прямой' или 'xN'
  • "effect" begins the line path-effect and is followed by path commands, and closed by
  • "endeffect"
  • "csurvey LAYER TYPE CATEGORY PEN" категория инструмента csurvey
  • "endsymbol"
Группа
Типы линий в одной группе могут быть соединены между собой при помощи "продолжения".

Это используется для указания того, что различные типы линий "стена" могут быть объединены.

ИНСТРУМЕНТ "ЛИНИЯ"

Инструменты "линий" заданы в текстовых файлах.

В программу встроены инструменты "линий": "пользовательский" и "стены". Остальные загружаются из подкаталога TopoDroid symbol/line. Этот каталог заполняется инструментами "линий", упакованными в дистрибутив APK. Вы можете удалить инструменты, которые вам не нужны, редактировать те, которые вы хотите изменить, и добавлять новые, не вошедшие в APK.

Файл инструмента "линий" содержит данные, определяющие свойства инструмента рисования.

  • "symbol line" [обязательный]
  • "name NAME", где NAME название линии по умолчанию [обязательный]
  • переводы названия, т.е. "name-es ..." [не обязательный]
  • "th_name NAME", где NAME это название в Therion, и должно совпадать с именем файла [обязательный]
  • "group GROUP" определяет, к какой группе принадлежит линия [не обязательный, по умолчанию - ни к какой]
  • "color rgb alpha" rgb и alpha цвет в шестнадцатиричном формате [rgb - обязалельный параметр, параметр альфа - не обязателен]
  • "width SIZE" множитель "базовой" толщины линии [не обязательный]
  • "dash PATTERN" рисунок штриховки [не обязательный]
  • "level" view_flag [optional, 1 if missing]
  • "style STYLE" где STYLE может быть 'прямой' или 'xN'
  • "effect" начало кода, определяющего рисунок линии и последующих команд.
  • "endeffect" закрывает секцию кода линии
  • "csurvey LAYER TYPE CATEGORY PEN" категория инструмента csurvey
  • "endsymbol"
Группа
Типы линий в одной группе могут быть соединены между собой при помощи "продолжения".

Это используется для объединения различных типов линий «стена».

Эффект рисунка
Этот параметр используется для определения линий со сложным стилем.

Слой
Атрибут "level" считывает значение флага для того чтобы отображать область только на определенном слое.
Флаг это сумма чисел 1 (базовый), 2 (пол), 4 (отложения), 8 (потолок) и 16 (артефакты).

План проекта пещеры

Окно План проекта пещеры отображает выбранные съемки проекта пещеры в проекции на горизонтальную плоскость (тоесть в плане).

Изображение можно перемещать и увеличивать/уменьшать.
Выбрав пикет одной съемки (нажав на ее ноиер), эту съемку можно перетаскивать, оствляя остальные съёмки в неподвижном состоянии.

Таким способом, пикет можно приблизить к пикету другой съемки.
Два пикета можно выбрать одновременно, касанием экрана рядом с ними и «отождествить/связать» два пикета друг с друом и добавить в проект пещеры.

Когда два пикета «приравнены», они соединяются пунктирной линией на плане.

Соедитять съёмки также можно вручную через Диалог соединения. Список соединений отображается в Диалоге соединения, через который их можно удалить из проекта пещеры.

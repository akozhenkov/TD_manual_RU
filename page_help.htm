МЕНЮ ПОМОЩИ

Диалоговое окно короткой справки открывается из меню «Справка» окон TopoDroid.

Оно содержит краткое описание функций кнопок и меню.

Кнопка «книга» в правом верхнем углу открывает руководство пользователя на разделе о текущем окне. Страница руководства пользователя может быть открыта также с помощью кнопки меню телефона Android.

Длительное нажатие на кнопку «книга» открывает руководство пользователя в первом разделе.

ИНФОРМАЦИЯ DistoX A3

Это диалоговое окно открывается нажатием кнопки Инфо в окне Приборов.

DistoX A3 Информация:

  • код устройства
  • единицы измерения угла
  • компас и клинометр вкл/выкл
  • обычный/калибровочный режимы
  • беззвучный вкл/выкл
В диалоге есть кнопка очистки памяти DistoX , т.е. зануления горячего бита который отмечает данные для отправки. После очистки DistoX должен быть выключен и снова включен для того чтобы изменения возымели эфект.

Смена модели прибора нужна для того, чтобы TopoDroid использовал подходящий протокол комуникации с DistoX и верно нитерпретировал данные.
По умолчанию должна быть выбрана подходящая модель.

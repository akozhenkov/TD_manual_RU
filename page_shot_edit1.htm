Shot edit (1)

This dialog is opened tapping a shot entry in the Shot list window.

It displays

  • length, azimuth, inclination. Editable for manually entered shots; not for downloaded shots
  • differences from the average acceleration, magnetic field, and magnetic dip
  • shot stations
  • left arrow: load previous leg
  • reverse: swap stations
  • right arrow: load next leg
  • shot comment
  • profile "extend" choice: "left", "vertical", "right"
  • six check buttons:
    duplicate and surface: shot flags
    Repeated leg: the shot belongs to the preceeding leg.
    Next leg: the shot belogs to the following leg.
    Splay group: apply the station change to all splays in the group of the shot.
    Renumber: renumber shots from this onward.
  • OK: save the changes and close the dialog.
  • Save: save the changes without closing the dialog.

Station cut-n-paste
Station names can be 'cut', 'copied', and 'pasted' across the app, in any dialog where there is a station entry field.

Long-tapping the station entry field opens the drop-down menu with the three options

  • cut
  • copy
  • paste

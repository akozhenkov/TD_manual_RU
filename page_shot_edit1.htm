ИЗМЕНЕНИЕ ПАРАМЕТРОВ ЗАМЕРА (1)

Это диалоговое окно открывается при нажатии на строку замера в окне Списка пикетов.

Оно отображает

  • длину, азимут, угол:. Редактируемые для пикетов, введенных вручную; НЕ редактируемые для пикетов, скачанных с DistoX
  • отклонения ускорения "A", магнитного поля "М" и угла наклона магнитных силовых линий "D" от соответствующих средних значений, только для измерений DistoX2
  • дата и время замера
  • пикеты: замера
  • стрелка влево: загрузить предыдущий замер
  • развернуть: поменять пикеты местами
  • стрелка вправо: загрузить следующий замер
  • комментарий
  • направление "развертки" разреза: "лево", "вертикально", "право"

Для замеров BRIC дата и время — это значения, загруженные вместе с данными замера с устройства. Для других устройств и для замеров, введенных вручную, это время загрузки/создания.

Кнопки:

  • Дубликат: замер, дублирующий ранее выполненное измерение.
  • Поверхностный: замер, сделанный на дневной поверхности.
  • измерение закоментирвано отключено.
  • Предыдущий замер: измерение принадлежит к пердыдущему вектору между пикетами.
  • Следующий замер: измерение принадлежит к следующему вектору между пикетами.
  • замер сделанный на предыдущий пикет.
  • Перенумеровать: перенумеровывает этот замер и все последующие, в соответствии с текущей политикой автонумерации пикетов. Дожны быть указаны оба номера пикета: ОТ и ДО.
  • Группа бок. замеров: изменяет номер пикета для всей группы боковых замеров этого пикета.
  • Следующая кнопка скрывает отображение боковых замеров на плане, разрезе или везде. (только сплеи)
  • X-Splay убрать (или присвоить) класс сплея для этого замера и предыдущих сплеев. X-splays показаны зеленым цветом. X-splays поддерживаются cSurvey.
  • OK: сохранить изменения и закрыть диалоговое окно.
  • Сохр: сохранить изменения без закрытия диалогового окна.
  • Больще/меньше: показать/скрыть дополнительные действия [A]
  • Отмена: закрывает диалог без сохранения

Такие флаги замеров, как "дубликат", "поверхностный" и "комментированный" являются взаимоисключающими. Замеры с флагом не учитываются в статистике съёмки. Экспорт замера также зависит от присвоенго флага. Замер экспортируется либо с флагом, если формат его поддерживает, либо как комментарий.

Появляется предупреждение после редактирование замера, в случае если он отличается от другого замера на тех же пикетах более чем на четверть длины или 15°.

Вырезать - вставить пикет
Номера пикетов могут быть 'вырезаны', 'скопированы', и 'вставлены' внутри приложения, в любом диалоге, где есть поле для номера пикета.

Длинное нажатие на поле ввода пикета открывает выпадающее меню с тремя опциями:

  • вырезать
  • копировать
  • вставить
Переназначение пикетов
Последоватеотное пререназначение (перенумерация) номеров пикетов может быть инициированна только из замера между пикетами. По умолчанию перенумерация применяется ко всем последующим замерам до конца списка. В случае если опция splay station выключена, номера пикетов для сплеев не изменятся.

Обычное переназначение следует использовать только для самых последних нескольких замеров, на случай если замер не был обнаружен (т.е. только два измерения), или ошибочно обнаружен (очень близкие сплеи).

Больше контроля над замерами, которые переназначаются, достигается с помощью выбора «нескольких замеров». где пользователь может выбрать замеры для переназначения по отдельности.

Splay group
A group of splays consists of all the splays between two legs. Splay group station change applies only to the contiguous set of splays with same station as the edit shot. Only FROM station must have been specified.

If the custom setting "Splay group" is set, only the splays after the editing shot are modified.

Классы сплеев
Классы сплеев включаются в разделе " спец настройки". Существует четыре класса:
  • обычные, обычные сплеи. Голубым цветом
  • сечений сплеи поперечных сечений. Зеленым цветом
  • горизонтальные, сплеи на плане. Темно синим цветом
  • вертикальные, сплеи на разрезе. Фиолетовым цветом
Кнопка Класс сплеев перключает межды четыремя классами сплеев. Замер сплея устанавливается на выбранный класс, когда данные сохраняются. Если опция «все сплеи» отмечена, все смежные предыдущие сплеи того же класса превращаются в "обычный".

> Классы сплеев устанавливаются с помощью кнопки выбора множественных действий "специальные действия". В множественном выборе вы можете выбрать несколько сплеев и назначить им всем одинаковый класс.

Если вы снимаете сплеи в одинаковой воследовательности на всех пикетах, классы сплеев можно настроить, редактируя замер включив кнопку группы сплеев. Сначала вы должны измерять сплеи для сечения по начиная с +90 и далее по кругу. Потом вы замеряете Гор-сплеи. Наконец, вы замеряете сплеи в вертикальной плоскости, начиная снова с +90.

Дополнительные действия
Диалог отображает номера пикетов и данные измерений (длину, азимут, угол).

При уровнях функционала "Базовый" или "Нормальный" в этом диалоге будет одна кнопка: Удалить измерение.

При более высоких уровнях функционала в диалоге будет семь кнопок, слева на право:

  • диалог Фото
  • Аудио заметка
  • диалог Измерения при помощи датчиков телефона
  • Добавить: замер перед текущим (введя данные измерений вручную)
  • Разделить: съёмку на этом пикете (создается новая съёмка)
  • Удалить замер
  • пометить измерение как проверку калибровки

Кнопка Удалить не удаляет замер из базы данных. Замер помечается как "удаленный" и может быть восстановлен позже через диалогВосстановления. Если замер один из серии измерений на этом отрезке, то вы можете удалить все измерения, поставив галку в соответствующее поле под кнопками.

Для замеров, также есть кнопка калибровочный. При помощи этой кнопки замер помечается как "проверка-калибровки". Данные измерения не используются при съёмке, измерения должны выполняться с разными углами разворота вокруг оси прибора. Эти замеры заносятся в базу данных и несут в себе информацию о качестве калибровки прибора в момент выполнения съёмки. Они могут быть показаны через диалог восстановления удаленных замеров.

Кнопка Разделить съёмку.
Она открывает диалоговое окно для ввода информации о новой топосъёмке. После чего данные съёмки будут перенесены в новую топосъёмку, начиная с этого пикета. Затем программа продолжает в новой съёмке. Возможно, вам придется разделять съёмки, когда у вас есть данные из двух (отдельных) частей пещеры, хранящихся в приборе: они загружаются в одну съёмку, которая затем должна быть разделена соответствующим образом.

Внимание. Разделение съёмки это необратимый процесс. TopoDroid не поддерживает сшитие съёмок, поскольку для этого необходимо очень большое кол-во действий. Настоятельно рекомендуем сдлеать бекап в zip перед тем как разделить съёмку. Из бекапа можно будет восстановить оригинал в случае неудачи.

ЛПВН на пикете
Также в этом диалоге можно вручную задать расстояния лево, право, верх и низ на одном из пикетов этого замера. Замеры ЛПВН на пикете ОТ или ДО могут быть вписаны в четыре соответствующих поля и сохранены нажатием на кнопку "Добавить ЛПВН". Пикет, с которого заданы расстояния, можно вбрать включив его над полями для ввода расстояний.
Также возможно добавит "промежуточные" ЛПВН задав расстояние вдоль замера, между точкой ЛПВН и пикетом ОТ.
В случае с боковыми замерами можно добавить расстояния только к пикету ОТ.
"Промежуточные" ЛПВН добавляются в список пикетов, другие добавляются перед замером.
Введенные вручную ЛПВНD будут помечены как "x-splays".

Splay color
Для сплеев также есть кнопка для назначения цвета замеру.
Список замеров не сразу обновляется цветами. Чтобы отобразить цветное изображение, нажмите кнопку «Обновить».

Survey notes

This dialog is opened tapping the Notes button of the Shot list window, the Survey info window, and the Sketch window (in "moving" mode).

With this dialog you can write and edit a generic text with notes that integrates the data and the drawing of your survey.

  • Save: save the changes and close the dialog
  • Back key: close the dialog without saving the changes

The notes are saved in a text file in the "txt" subdirectory of "TopoDroid".
You can also edit this file with other text editors.

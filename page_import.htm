ИМПОРТ СЪЁМКИ

Это диалоговое окно открывается касанием кнопки Импорт (стрелка вниз) в главном окне.

Оно отображает список zip архивов в каталоге zip, которые могут быть импортированы в приложение.

Отображает список поддерживаемых форматов файлов импорта: dat (Compass), svx (Survex), th (Therion), tro и trox (VisualTopo), csn (CaveSniper), top (PocketTopo), и TopoDroid zip архивы.
Импортируемый файл может быть выбран через системный проводник.

Для некоторых форматов можно настроить импорт с помощью нескольких опций.
Нажмите на имя архива, чтобы импортировать съемку. Если имя файла съёмки, которую вы импортируете, уже существует в базе данных, файл не импортируется.

Опции импорта Compass (dat)

  • LRUD если выбрана опция LRU, то данные импортируются и преобразуются в сплеи
  • Сначала замеры если включено, то ЛПВН и сплеи импортируются после замеров
  • Режим подводной съёмки включите для импорта подводной топосъёмки в формате Compass [T]

VisualTopo (tro, trox) import options

  • ЛПВН если включено, то ЛПВН будут импортированы из файла и сконвертированы в сплеи
  • Сначала замеры если включено, то ЛПВН и сплеи импортируются после замеров
N.B. Импорт может завершиться неудачно (с аварийным закрытием приложения) из-за сбоя в базе данных TopoDroid. Если это произойдет, удалите частично импортированную съёмку и повторите попытку импорта.

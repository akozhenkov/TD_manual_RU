Импорт съёмки

Это диалоговое окно открывается нажатием на кнопку Импорт (стрелка вниз) в главном окне.

Оно отображает список файлов, которые могут быть импортированы в приложение.

TopoDroid может импортировать файлы топосъёмки в следующих форматах:

  • Therion: ".th" файлы нитки хода в простом формате ("normal from to length compass clino" и опционально LRUD). Therion scrap файлы не импортируются.
  • Compass ".dat" файлы (не полностью оттестировано)
  • VisualTopo ".tro" файлы (не полностью оттестировано)
  • PocketTopo ".top" Файлы. Импортируются и данные, и абрисы (не полностью оттестировано)
  • CaveSniper ".csn" текстовые файлы.
  • TopoDroid ZIP архивы. Это специальные zip архивы, в которых есть вся информация для того, чтобы создать проект, абрисы (".th2" формат), заметки и фото.

Zip архивы импортируются из папки "zip" (из той же, куда TopoDroid их сохраняет). Файлы съёмок импортируются из папки "import". Поэтому вы должны скопировать файл съёмки, который вы хотите импортировать в этот каталог. Если имя файла съёмки, которую вы импортируете, уже существует в базе данных, файл не импортируется.

Импорт Therion
Только файлы Therion с форматом данных "normal" могут быть импортированы.
TopoDroid не может импортировать скрапы Therion scraps.

Импорт PocketTopo
TopoDroid может импортировать данные съёмки и абрисы. При импорте из PocketTopo каждый цвет соответствует определенному типу линий и точек TopoDroid. Соответствие цветов и объектов может быть изменено в настройках.

Импорт Compass
TopoDroid може конвертировать данные из формата "normal" в формат "diving", в случае если включена соответствующая опция [T].

N.B. Импорт может завершиться неудачно (с аварийным закрытием приложения) из-за сбоя в базе данных TopoDroid. Если это произойдет, удалите частично импортированную съёмку и повторите попытку импорта.

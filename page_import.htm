Импорт съёмки

Это диалоговое окно открывается нажатием на кнопку Импорт в главном окне.

Он отображает список файлов, которые могут быть импортированы в приложение.

TopoDroid может импортировать файлы съемки в следующих форматах:

  • Therion: ".th" файлы нитки хода в простом формате ("normal from to length compass clino" и опционально LRUD). Therion scrap файлы не импортируються.
  • Compass ".dat" файлы (не полностью оттестировано)
  • VisualTopo ".tro" файлы (не полностью оттестировано)
  • PocketTopo ".top" Файлы. Импортируються и данные и абрисы (не полностью оттестировано)
  • TopoDroid ZIP архивы. Это специальные архивы zip в которых есть вся информация для того чтобы создать проект, абрисы (".th2" формат), заметки и фото.

Zip архивы импортируються из папки "zip" (из той же, куда TopoDroid сохраняет их). Файлы съёмок импортируються их папки "import". Поэтому вы должны скопировать файл съёмки, который вы хотите импортировать в этот каталог. Если имя файла съёмки, который вы импортируете уже существует в базе данных файл не импортируется.

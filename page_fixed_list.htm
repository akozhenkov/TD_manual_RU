Это диалоговое окно открывается нажатием на кнопку GPS в окне Информация о съёмке.

Диалог показывает список привязанных пикетов, и содержит три кнопки, чтобы создать новую точку:

  • GNSS определяет местоположение при помощи GPS телефона (если выставлено разрешение на доступ к местоположению)
  • Добавить ввести координаты вручную
  • Импорт координат из GNSS app (только для версий ОС Android ниже 10)
  • Запустить приложение Gps Test

Внешние приложения GNSS
TopoDroid can import geopoint coordinates from

  • MobileTopographer файлы с точками. Откроется диалоговое окно для выбора геоточки.
  • GPX recorder трек (gpx) файл. Используется последняя точка трека.
  • GPX position файл (csv) с точками. TopoDroid вычисляет среднее значение позиций.
  • GPS Test позиция из лог файла. TopoDroid использует последнюю позицию.

Номер пикета геоточки, импортированное из GPX Recorder, GPX Position, and GPS Test устанавливается в TopoDroid автоматически как номер с префиксом '#'.

MobileTopographer файлы списка точек хранятся в папке MobileTopographer/pointlists.

GPS Recorder файлы треков хранятся в папке gpx_recorder.

GPS Position файлы хранятся во внешнем хранилище.

GPS Testведение журнала должно быть включено в настройках приложения. Достаточно зафиксировать координаты. Файл журнала сохраняется при закрытии приложения и хранится в папке gnss_log.
Кроме того, координаты геоточки сохраняются в буфере обмена с GpsTest app и могут быть переданы в поля в диалоговом окне вставка. Обязательно установите флажок для сохранения высот в буфере обмена.

Файлы отслеживания GPS Logger должны быть экспортированы в формате "csv". TopoDroid использует высоту эллипсоида и точность.

GNSS советы (поищите в инете подробности и как сделать)

  • Включить «принудительное полное измерение GNSS»
  • Если доступно, отключите «кинематический фильтр Калмана».
  • Если доступно, установите местоположение "статический режим"
  • При наличии отключите "A-GPS". Отключите «мобильные данные» и очистите данные A-GPS.
  • Отключить местоположение "исправление wi-fi и bluetooth"

Примечание. Отключение оптимизации батареи для приложения необходимо только в том случае, если вы хотите предотвратить система закрывает приложение, когда оно не на переднем плане.

Настройки местоположния

  • Формат координат: либо dd.mm.ss либо доли градусов [A]
  • C.R.S. система координат [по умолчанию географические координаты] (выключено)
  • Отрицательные высоты по умолчанию нет]
  • Редактируемые высоты [по умолчанию нет]
  • Время осреднения местопложения [по умолчанию 60с]
  • Geopoint app [по умолчанию нет]

Locations list

This dialog is opened tapping the GPS button in the Survey info window.

The dialog shows the list of locations, and has three buttons to create a new point:

  • GPS gets the position with the GPS
  • Add enter the coordinates manually
  • Import the position from Mobile Topographer
Tapping a location entry in the list opens the Location edit dialog.

Internally TopoDroid stores the coordinates in decimal degrees. In the coordinate dialogs you can type them either in decimal degrees or as "dd:mm:ss.ss". The TopoDroid keypad has special buttons for "degree" and "minute". If you use a keyboard that does not have semicolon ':', use the space instead. If the keyboard does not have the decimal point '.' use the slash '/' instead.

Location settings

  • Location units: either dd.mm.ss or decimal degrees [A]
  • C.R.S. coordinate reference system [default is long-lat] (disabled)

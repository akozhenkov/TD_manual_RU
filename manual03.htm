[3] ОКНО УСТРОЙСТВ

Перед началом съемки необходимо подключить и выполнить сопряжение Android устройства с DistoX и откалибровать его. Нажмите кнопку DistoX в главном окне, чтобы открыть окно устройств. Будет показан список известных устройств DistoX. TopoDroid единовременно работает только с одним DistoX. Чтобы выбрать прибор, нажмите на его строку в списке. Имя выбранного DistoX отображается в верхней части экрана, чуть ниже кнопок. Если не выбран ни один DistoX, то вы увидите красное сообщение "Устройство не выбрано".

Если список DistoX пуст, ваш DistoX должен быть подключен и сопряжен с Android. Это может быть сделано при помощи TopoDroid (через меню Сканировать и Сопряжение) либо через Настройки телефона.

Псевдоним DistoX
По умолчанию TopoDroid использует адрес BT в качестве имени для DistoX. Разным DistoX можно давать псевдонимы, чтобы легко различать несколько устройств.

Внимание. Кнопки помеченные [*] работают только при подключенном DistoX.

ДЕЙСТВИЯ

  • Нажатие на строку DistoX выбирает текущее устройство.
  • Длинное нажатие на строку DistoX: устанавливает псевдонимы

КНОПКИ

При нажатии на любую кнопку кроме первой и третьей, необходимо, чтобы DistoX был подключен .
  • Bluetooth: сброс соединения.
  • Переключить: DistoX в режим калибровки и обратно. [*]
  • Калибровка: текущего устройства.
  • Память: DistoX, манипуляции с памятью. [*]
  • Считать [A]: и отобразить калибровочные коэффициенты из DistoX. [*]
  • Информация [A]: некоторая информация о DistoX. Отображаемая информация будет разной для приборов DistoX A3 и DistoX X310 [*]

МЕНЮ

  • Сканировать: BT окружение в поисках DistoX. DistoX должен быть включен. Вам, возможно, придется попробовать два или три раза, прежде чем DistoX будет найден.
  • Сопряжение: текущего DistoX с устройством Android, в случае если еще не сопряжено.[*]
  • Отсоединить: текущее устройство (DistoX останется сопряжен)
  • Прошивка: Скачать или загрузить прошивку с\на DistoX2 [*]
  • Настройки: Настройки устройства и данных калибровки.
  • Помощь Описание кнопок и пунктов меню.

НАСТРОЙКИ УСТРОЙСТВА

  • Bluetooth: следует ли включить, выключить или проверить при запуске [по умолчанию "проверять"]
  • Типы подключения : для передачи данных. Можно оставаться на связи с DistoX постоянно чтобы скачивать данные замеров, как только они будут сделаны, либо скачивать данные скопом, "по требованию". В режиме "мульти" все также как и в "по требованию", но долгое нажатие на кнопку позволяет переключаться между DistoX. Это относится только к данным: другие функции DistoX используют только ручной режим скачивания [по умолчанию "по требованию"]
  • Авто подключение: автоматическое повторное подключение, если соединение теряется в "постоянном" режиме загрузки данных [по умолчанию "нет"]
  • Кол-во новых данных: выяснить кол-во новых замеров перед тем как скачать их (режим соединения по требованию) [по умолчанию "нет"]
  • Тип соединения: тип подключения - нормальной или "небезопасный" [по умолчанию "нормальный"]
  • Решение для Z6: Устранение ошибки загрузки из-за тайм-аута на телефоне Z6, и, возможно, других устройствах. Можно оставить включенным [по умолчанию "вкл"]
  • Автоматическое сопряжение: следует ли TopoDroid попытаться выполнить сопряжение с DistoX, выбранным при сканировании [по умолчанию - "нет"]
  • Настройки калибровки: открывает меню настроек данных калибровки

Спец настройки: Задержка подключения, Пауза скачивания, Пауза ожидания готовности, Пауза вкл. лазера, Пауза после замера.

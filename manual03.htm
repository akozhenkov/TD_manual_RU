[3] ОКНО УСТРОЙСТВ

Перед началом съемки необходимо подключить и выполнить сопряжение DistoX с Android устройством и откалибровать его. Тапните кнопку DistoX на главном экране, чтобы открыть окно устройств. Будет показаг список известных устройств DistoX. TopoDroid единовременно работает только с одним DistoX. Чтобы выбрать прибор нажмите на его строку в списке. Имя выбранного DistoX отображается в верхней части, чуть ниже кнопок. Если не выбран ни один DistoX, то вы увидите красное сообщение "Устройство не выбрано".

Если список DistoX пуст, ваш DistoX я должен быть подключен и сопряжен с Android. Это может быть сделано при помощи TopoDroid (Сканировать и Сопряжение) либо через Настройки телефона.

Псевдоним DistoX
По умолчанию TopoDroid использует адрес BT в качестве имени для DistoX. Разным DistoX можно давать псевдонимы чтобы легко различать несколько устройств.

ДЕЙСТВИЯ

  • Нажатием на строку DistoX выбирается текущее устройство.
  • Длинное нажатие на DistoX устанавливает псевдонимы

КНОПКИ

Необходимо чтобы DistoX был включен при нажатии на любую кнопку кроме первой и третьей.
  • Bluetooth сброс соединения. Если не установлено соединение с DistoX.
  • Переключить DistoX в режим калибровки и обратно.
  • Калибровки текущего устройства.
  • Память DistoX, монипуляции с памятью.
  • Счтать [A] и отобразить калибровочнын коэффициенты из DistoX.
  • Информация [A]: некоторая информация о DistoX

МЕНЮ

  • Сканировать BT окружение в поисках DistoX. DistoX должен быть включен. Вы, возможно, придется попробовать два или три раза, прежде чем DistoX будет найден.
  • Сопряжение текущего DistoX с устройством Android, в случае если еще не сопряжено.
  • Очистить текущее устройство (DistoX останется сопряжен)
  • Скачать или загрузить прошивку с\на DistoX2
  • Настройки
  • Помощь

НАСТРОЙКИ УСТРОЙСТВА

  • Bluetooth: следует ли включить, выключить или проверить при запуске [по умолчанию "проверять"]
  • Подключение для передачи данных: либо оставаться на связи с DistoX чтобы скачивать данные замеров как только они будут сделаны, либо скачивать данные скопом, "по требованию". Это относится только к данным: другие функции DistoX используют только ручной режим скачивания [по умолчанию "по требованию"]
  • Автоматическое переподключение: автоматическое повторное подключение, если соединение теряется в "непрерывном" режиме загрузки данных [по умолчанию "нет"]
  • Сокет тип подключения: нормальной или "небезопасный" [по умолчанию "нормальный"]
  • Z6-workaround Устранение ошибки загрузки из за тайм-аута на телефоне Z6, и, возможно, другизх устройствах. Можно оставить включенным [по умолчанию "вкл"]
  • Задержка соединения: задержка [1/10 сек], прежде чем пытаться подключиться к DistoX [по умолчанию 0, отсутствие задержки]
  • Автоматическое сопряжение: следует ли TopoDroid попытаться выполнить сопряжение с DistoX, выбранным из сканирования [по умолчанию "нет"]

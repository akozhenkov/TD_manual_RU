ВСПОМОГАТЕЛЬНЫЕ ПРИЛОЖЕНИЯ

При работе с TopoDroid, вам могут быть полезны некоторые вспомогательные программы:
  • ThManager позволяет сшивать съёмки, экспортированные в формат Therion, в единый проект. Т.Е. создает файл thconfig с командами equate.
  • TdManager это альтернатива ThManager. Он соединяет съемки непосредственно из базы данных TopoDroid, не прибегая к экспорту
  • Proj4 это очень простой конвертор координат, он нужен TopoDroid для конвертации широты и долготы в другие системы координат
  • MobileTopographer приложение для определения местоположения. С помощью него вы можете определять координаты точки и считывать их в TopoDroid из файла с точками MobileTopographer.
  • GPX Recoder и GPS Position может сохранять треки геоточек в файлы gpx и csv соответственно. TopoDroidможет читать эти файлы и использовать последнюю геоточку для установки координат фиксированного пикета.
  • GPS Test можно поместить координаты в буфер обмена, откуда их можно вставить в TopoDroid.
  • CrowdMag это приложение Национального центра геофизических данных NOAA, полезное для получения информации о текущем состоянии магнитного поля Земли.
  • Приложение для фото (обычно стандартное приложение Камера)
  • Файл менеджер (например. TotalCommander) нужен для просмотра файлов и создания резервных копий.
  • SQLiteViewer (либо другой SQLite менеджер) нужен для просмотрка базы данных TopoDroid
  • Текстовый редактор (например. Jota), чтобы можно было просматривать текстовые файлы

Кроме того, в зависимости от того как вы обрабатываете ваши съёмки, вы можете установить:

  • Редактор и просмотрщик изображений (обычно просмотрщик предустановлен в системе, например Галерея)
  • Редактор и просмотрщик DXF
  • Редактор и просмотрщик таблиц (например. Google Sheets)
  • Просмотрщик SVG (например. SVG viewer)
  • Архиватор ZIP (ZArchiver)
  • GoogleEarth
  • Просмотрщик GPS треков (например OSMand)

Proj4
TopoDroid хранит координаты точек в географической системе координат long-lat WGS-84. Высоты отсчитываются как по отношению к эллипсоиду, так и к геоиду.

Приложение Proj4 используется TopoDroid для преобразования координат между WGS-84 и другими системами координат.

  • для пересчета координат точки из long-lat (WGS-84) в другую систему координат.
  • для ввода координат точки в другой системе координат.

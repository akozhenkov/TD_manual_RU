3D VIEWER - Model inf

Диалог 3D-информация открывается из меню Инфо в окне 3D-визуализации.

Он представляет некоторую общую информацию

  • Размер ячеек сетки
  • Текущее направление взгляда
  • Количество съемок
  • Количество замеров и сплеев
  • Количество пикетов
  • Общая длина
  • Общая глубина
  • Предполагаемый объем (если модель была рассчитана со стенами)
  • Протяженность по долготе
  • Протяженность по широте
  • Диапазон высот

Далее следует список съемок, составляющих модель. При нажатии на съемку появляется диалоговое окно с количеством замеров и сплеев в съемке и их длина.

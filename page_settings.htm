Общие настройки TopoDroid

НАСТРОЙКИ
  • Рабочий каталог. По умолчанию "не задан" (т.е. используется "TopoDroid")
  • Размер текста: размер текста в списках >= 1. По умолчанию - 16
  • Размер кнопок {маленькие, нормальные, средние, большие, огромные}. По умолчанию нормальные
  • Уровень функционала {базовый, нормальный, продвинутый, экспертный}. По умолчанию нормальный
  • Клавиатура: Использовать стороннюю клавиатуру вместо системной
  • Совместная съёмка: Сервис совместной топосъёмки. По умолчанию ВЫКЛЮЧЕН. Если функция совместной съёмки не активирована, то этот пункт не отображается.
  • Язык {default,en,es,fr,de,it,ru,bg и др.}. По умолчанию - "default" (язык системы)

ИМПОРТ-ЭКСПОРТ

  • Преобразование цветов PocketTopo: Назначить соответствие цветов PocketTopo типам объектов TopoDroid при импорте данных PocketTopo.
  • Оценить размер Лево/Право при импорте Compass/VisualTopo, По умолчанию ВКЛ.
  • Формат экспорта данных {th,svx,dat,srv,top,tro,dxf,svg,csv} По умолчанию therion
  • Формат экспорта абрисов по умолчанию {отсутствует, Therion, cSurvey, DXF, SVG, PNG} По умолчанию therion
  • Карты Therion в первую очередь, до данных топосъёмки. По умолчанию нет
  • Точки пикетов Therion. Добавлять точки пикетов на скрапы Therion. По умолчанию да
  • Боковые замеры в Therion. Добавлять линии боковых замеров на скарапы Therion. По умолчанию нет
  • Экспортировать префиксы пикетов. Экспорт номеров пикетов с префиксами для формата cSurvey.
  • Экспортировать префиксы пикетов в Compass. По умолчанию нет
  • Боковые замеры в Compass: Добавлять линии боковых замеров в файлы Compass
  • Номера пикетов Toporobot. Экспорт номеров пикетов в стиле Toporobot.
  • Мин. угол бок. замера для ЛПВН. Минимальный угол между вектором и боковым замером для расчета ЛПВН при экспорте. По умолчанию 0
  • ВН минимальный наклон боковых замеров. По умолчанию 0
  • ЛП максимальный наклон боковых замеров. По умолчанию 90
  • Развернуть ЛП в Compass. Поменять местами Лево и Право при экспорте в формат Compass при экспорте. Необходимо для корректной работы при съёмке обратными замервми. По умолчанию выкл.
  • Окончание строки для Survex/Topo {Linux,Windows}. По умолчанию Windows
  • Конечный пикет луча Survex: добавлять пикет ДО в боковые замеры Survex. По умолчанию нет
  • Survex ЛПВН. Добавить ЛПВН при экспорте в формат Survex. По умолчанию нет
  • Расстояние между точками: расстояние между точками в линии при экспорте линий
  • Сетка в SVG. Добавить сетку в экспорт SVG. По умолчанию выкл.
  • SVG бергштих направления линий. По умолчанию выкл
  • SVG размер точек. По умолчанию 0.1
  • SVG размер номеров пикетов. По умолчанию 0.3
  • SVG толщина линий. По умолчанию 0.5
  • SVG толщина линий сетки. По умолчанию 0.5
  • SVG толщина линии нитки хода. По умолчанию 0.5
  • SVG толщина линии направления. По умолчанию 2.0
  • Пикеты в KML/GeoJSON. Добавить точки пикетов при экспорте данных в формат KML. По умолчанию выкл.
  • Боковые замеры в KML. Добавить линии боковых замеров при экспорте данных в формат KML. По умолчанию выкл.
  • Масштаб PNG. Масштабный коэффициент абриса при экспорте в PNG, [0.5, 100]. По умолчанию 1.5 [x200 px]
  • Цвет фона PNG. Цвет фона PNG при экспорте абрисов. (RGB in 0-100). По умолчанию "0 0 0" [черный]
  • Точки бок. замеров в DXF. Экспортировать конечные точки боковых замеров в файл DXF
  • Версия AutoCAD {6,12}. По умолчанию 6

  • Порог вертикальности для отрезков: Вертикальный порог для отрезков [0,80]. Максимальный наклон отрезков после которго ЛПВН переходит в ЗВСЮ По умолчанию 80 градусов

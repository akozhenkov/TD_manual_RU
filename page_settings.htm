Настройки TopoDroid

ОБЩИЕ НАСТРОЙКИ
  • Рабочая директория. По умолчанию "не задана" (т.е. используеться "TopoDroid")
  • Размер текста >= 1. По умолчанию 14
  • Размер кнопок {маленькие, большие}. По умолчанию маленькие
  • Уровень функционала {базовый,нормальный,продвинутый,эксперементальный}. По умолчанию нормальный
  • Сервис совместной топосъёмки. По умолчанию ВЫКЛ
  • Язык {en,es,fr,de,it,ru,bg}. По умолчанию "не задан" (язык системы)

НАСТРОЙКИ СЪЁМКИ

  • Группа. По умолчанию "не задано"
  • Порядок наименования пикетов {нет, ЛПВН+след пикет, ЛПВН+пред пикет, след пикет+ЛПВН, пред пикет+ЛПВН, пред пикет+ЛПВН+след пикет}. По умолчанию "ЛПВН+след пикет".
  • Номера пикетов {буквенно-цифровые,цифровые}. По умолчанию буквенно-цифровые
  • Номер начального пикета. По умолчанию "не задано"
  • Размер миниатюры фото [80,400]. По умолчанию 200 пикселей
ЕДИНИЦЫ ИЗМЕРЕНИЯ
  • Единицы длинны {метры, футы}. По умолчанию метры
  • Единицы углов {градусы, грады}. По умолчанию градусы
  • Единицы шага сетки {метры, ярды}. По умолчанию метры
ПИКЕТЫ
  • Близость замеров между собой > 0 [проценты]. По умолчанию 0.05 (5%)
  • Минимальное число замеров векторе >=2. По умолчанию 2
  • Порог для развертки боковых замеров "вертикальный" [0,90]. По умолчанию 10 градусов
  • Порог для развертки вертикальной развертки векторов [0,80]. По умолчанию 80 градусов
  • Устанавливамый азимут "развертки". По умолчанию нет
  • Закрывать петли. По умолчанию нет
  • Поле для ввода обратного замера вручную. По умолчанию нет
  • Компенсация магнитных аномалий. По умолчанию нет
  • Таймер задержки измерения. По умолчанию 10 с
  • Звук таймера задержки измерения [10,100]. По умолчанию 50 %
МЕСТОПОЛЖЕНИЕ
  • Единицы координат {ггммсс, доли градусов}. По умолчанию ггммсс
ТОЧНОСТЬ
  • Ускорение. порог > 0. По умолчанию 400
  • Магнитизм. порог > 0. По умолчанию 300
  • Угловой порог > 0. По умолчанию 3 градуса

НАСТРОЙКИ АБРИСОВ

  • Панель выбор инструментов рисования {недавние,список,сетка}. По умолчанию недавние
  • Кол-во недавних инструментов {3, 4, 5, 6}. По умолчанию 4
  • Перетаскивание за край. По умолчанию нет
  • Кнопки зума {выкл,временно,постоянно}. По умолчанию постоянно
  • Проверять на не присоединенные пикеты. По умолчанию нет
  • Сохранять области в xtherion-"style". По умолчанию нет
ИНСТРУМЕНТЫ ТОЧКИ
  • Немасштабируемые символы точек. По умолчанию нет
  • Масштаб инструментов точек. По умолчанию 1.2
  • Размер символов. По умолчанию 24 пикселя
ИНСТРУМЕНТЫ ЛИНИИ
  • Толщина линий (толщина линий стен в два раза больше ). По умолчанию 1 пиксель
  • Тип линий {Безье,точные,нормальные,грубые}. По умолчанию нормальные
  • Минимальное расстояние между точками линии. По умолчанию 10 пикселей
  • Размер "штриха направления" у линий поперечного сечения [1,20]. По умолчанию 8 [x unit]
  • Видимость границы области. По умолчанию да
  • Продолжение линий {соединять,совмещать}. По умолчанию совмещать
  • Точность кривых безье. По умолчанию 1
  • Угловой порог кривых безье. По умолчанию 20
ЭЛЕМЕНТЫ АБРИСА
  • Толщинан нитки хода [0.5,10]. По умолчанию 1 пиксель
  • Размер номеров пикетов. По умолчанию 24 пикселя
  • Радиус зеленых точек [0.5,100]. По умолчанию 5 пикселей
  • Радиус выбора >= 0. По умолчанию 24
  • Порог угла для показа боковаого замера в плане [0,90]. По умолчанию 80 градусов
  • Порог угла для показа боковаого замера прерывистой [0,90]. По умолчанию 50 градусов
  • Порог угла для горизонтальных поперечных сечений [0,90]. По умолчанию 70 градусов
АВТО-СТЕНЫ
  • Тип стен {обычный,выпуклый}. По умолчанию обычный
  • Порог угла для боковаого замера в плане. По умолчанию 70 градусов
  • Порог угла для боковаого замера в разрезе. По умолчанию 45 градусов
  • Сепарация. По умолчанию 0.1 м
  • Вогнутость. По умолчанию 0.1
  • Шаг. По умолчанию 1 м

НАСТРОЙКИ ИМПОРТА/ЭКСПОРТА

  • Соответствие цветов PocketTopo
  • Рассчитыватьразвертку Л П при импорте, По умолчанию да
  • Формат экспорта по умолчанию {th,svx,dat,srv,top,tro,dxf,svg,csv} По умолчанию th
  • Добавлять пикеты на скарапы Therion. По умолчанию да
  • Префикс пикетов в cSurvey. По умолчанию нет
  • Окончание строки для Survex {Linux,Windows}. По умолчанию Linux
  • Добавлять пикет ДО в боковые замеры Survex. По умолчанию нет
  • Survex ЛПВН. По умолчанию нет
  • CSV единицы длинны {метры,футы}. По умолчанию метры
  • PNG разрешение [0.5, 10]. По умолчанию 1.5 [x200 пикселей]
  • PNG цвет фона (RGB in 0-100). По умолчанию "0 0 0" [черный]
  • DXF масштаб абриса. По умолчанию 1
  • AutoCAD версия {6,12}. По умолчанию 6

НАСТРОЙКИ УСТРОЙСТВА

  • Проверка блютуф при старте. По умолчанию да
  • Режим подключения {по требованию,постоянный}. По умолчанию по требованию
  • Авто переподключение (в непрерывном режиме). По умолчанию нет
  • BT тип подключения {нормальный,небезопасный}. По умолчанию нормальный
  • решение для Z6. По умолчанию да
  • Кол-во попыток подключения >= 0. По умолчанию 0
  • Автосопряжение. По умолчанию да

НАСТРОЙКИ КАЛИБРОВКИ

  • Политика группировки {4x4+по 1му,по 4ре,расстояние). По умолчанию по 4ре
  • Порог расстояния в группе. По умолчанию 40 [градусов]
  • Калибровка эпсилон. По умолчанию 0.000001
  • Максимальное кол-во итераций калибровки. По умолчанию 200
  • Показывать сырые калибровочные данные. По умолчанию нет
  • Алгоритм калибровки {авто,линейный,нелинейный}. По умолчанию авто

Конечная карта

В этом разделе описывается использование Therion или cSurvey, чтобы подготовить итоговую карту.

Помимо всего прочего, среди экспортированных данных вы найдете, заметки топосъёмки (текстовый файл), файлы абрисов (th2 файлы Therion), а также файл нитки хода (th Therion файл). В этом файле включены файлы абрисов, но они закомментированы. Для того, чтобы получить карту в экспортированных PDF файлах, вы должны раскоментировать эти строки. Это хорошая идея, добавить карту для каждого абриса (scrap); на самом деле две карты одна для плана, другая для разреза развертки.

Если вы хотите скомпилировать проект Therion, вы должны сделать thconfig файл, который может быть совсем простым, как например:

source my_survey.th
export map -proj plan -o my_cave_p.pdf
export map -proj extended -o my_cave_s.pdf

В качестве альтернативы вы можете работать в cSurvey. Откройте CSX файл с экспортированными абрисами в cSurvey: вы получите готовую таблицу, заполненную данными съёмки и абрисы, выровненные на нитку хода в cSurvey.

[14] ПРОЕКТЫ ПЕЩЕР

МЕНЕДЖЕР ПРОЕКТОВ ПЕЩЕР
Доступ к этому окну осуществляется с помощью кнопки Проекты в Главном окне.

Проект пещеры объединяет съёмки, которые находятся в базе данных TopoDroid. "Проекты пещер" хранятся в файлах, похожих на файлы Therion, в папке "tрconfig" каталога TopoDroid. Файл проекта сохраняется всякий раз, при закрытии проекта. Проекты можно экспортировать как файлы Therion или Survex; экспорт сохраняется в папки "th" и "svx", соответственно. Менеджер проектов TopoDroid отображает список пещерных проектов в папке "tрconfig". Список проектов будет пустым, если вы еще не создали ни одного проекта.

КНОПКИ

  • Новая проект: открывает диалог для создания нового проекта пещеры

МЕНЮ

  • Закрыть окно Cave Project Manager
  • Помощь отображает данную страницу

Касание строки с названием проекта открывает Окно проектов.

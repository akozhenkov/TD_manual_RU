[14] ПРОЕКТЫ ПЕЩЕР

МЕНЕДЖЕР ПРОЕКТОВ ПЕЩЕР
Доступ к этому окну осуществляется с помощью кнопки в Главном окне.

Проект пещеры объединяет съёмки, которые находятся в базе данных TopoDroid. "Проекты пещер" хранятся в файлах, похожих на файлы Therion, в папке "tdconfig" каталога TopoDroid. Файл проекта сохраняется всякий раз, при закрытии проекта. Проекты можно экспортировать как файлы Therion или Survex; экспорт сохраняется в папки "th" и "svx", соответственно. Когда вы запускаете TopoDroid Manager, он отображает список пещерных проектов в папке "tdconfig". Список проектов будет пустым, если вы еще не создали ни одного проекта.

КНОПКИ

  • Новая пещера: открывает диалог для создания нового проекта пещеры

МЕНЮ

  • Закрыть окно Cave Project Manager
  • Помощь отображает данную страницу

При нажатии на название проекта открывается Окно Проекта.

ПРОЕКТ ПЕЩЕРЫ
Окно проекта отображает съёмки, включенные в проект пещеры. Когда создается новый проект пещеры, в нем еще нет съёмок и список пуст. Слева от каждой съёмки есть чекбокс для выбора.

КНОПКИ

  • Добавить съёмки: открывает диалоговое окно для выбора съёмок из базы данных TopoDroid, которые должны быть включены в проект пещеры.
  • Удалить съёмки: удалить из проекта пещеры выбранные съёмки.
  • Показать съёмки: отображает выбранные съёмки в виде плана.
  • Список соединений между съемками проекта пещеры.
  • Просмотр проекта в 3D

МЕНЮ

  • Закрыть окно проекта пещеры
  • Экспорт проекта пещеры в форматы Therion или Survex
  • Удалить проект пещеры
  • Помощь показать данное руководство

Планы выбранных съёмок отображаются в Окне просмотра съёмок.

ОКНО ПРОСМОТРА СЪЁМОК
Съемки отображается в виде плана нитки хода и номерами пикетов. Каждая съёмка отображается своим цветом. Через некоторое время цвета будут повторятся, поэтому вам лучше отображать только две или три съёмки одновременно. Соединения показаны пунктирными красными линиями, а соединенные пикеты показаны в круге. Карту можно передвигать и увеличивать. Съемку можно переместить (нажать на пикет: он будет показан в круге и перетащить ее), в то время как другие съемки не перемещаются. Таким образом, вы можете совмещать пикеты из разных съемок, которые вы хотите соединить. Когда круги пикетов перекрывают друг друга обведите их, коснувшись точки: затем вы можете вставить соединение между ними с помощью меню "Добавить соединеиние".

КНОПКИ

  • Добавить соединение: открывает диалоговое окно, чтобы вручную добавить соединение между съемками проекта пещеры.
  • Список соединений между съемками проекта пещеры

МЕНЮ

  • Закрыть окно проекта пещеры
  • Помощь показать данное руководство

Вы можете удалить соединение, нажав на него в диалоговом окне, в котором перечислены соединения проекта пещеры.

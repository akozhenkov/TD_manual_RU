Листинг памяти

Это диалоговое окно открывается при нажатии на кнопку Дамп в диалоге Память DistoX

Он отображает содержимое части памяти DistoX.

Каждая строка содержит

  • адрес памяти
  • буквенный код который обозначает относится ли запись к нормальному измерению ('d') или калибровочному измерению ('g')
  • значения в памяти.

Записи без значений отмечены вопросительным знаком ('?').
Обычные замеры показаны как длинна (в метрах), азимут и угол (в градусах).
Показания "точности" замеров в DistoX2 не показываються.
Для данных калибровки показаны сырые данные (значения 'm' не показаны для DistoX2).

ТЕКУЩИЙ РАБОЧИЙ КАТАЛОГ

TopoDroid не предназначен для управления сложными проектами с большим количеством съёмок. Однако можно создать новый текущий рабочий каталог для хранения данных. Базовая рабочая директория TopoDroid помещается во встроенной памяти устройства, или, начиная с версии ОС Android 11 - public. До версии но при ОС Android 10 включительно эта директория может быть изменеа [T]. Каждый рабочий кататлог является подкаталогом в базовой директории, который создается автоматически. Имя текущего рабочего каталога должно начинаться с "TopoDroid". Если вы укажете имя, которое не начинается с "TopoDroid", то к нему будет добавлен префикс.

Символы для абрисов и файлы, связанные с приборами ( базы данных устройств, прошивки, дампы памяти, калибровки), содержаться только в каталоге по умолчанию "TopoDroid". Другие текущие каталоги содержат только рабочие файлы базы данных и файлы топосъёмки.

Вы можете использовать разные каталоги для проектов съемки разных пещер. В качестве альтернативы вы можете иметь один каталог "TopoDroid" для каждого проекта и переключиться на тот, в котором вы должны работать, путем переименования каталогов в файловом менеджере. Однако в этом случае общие файлы будут содержаться в каждом из каталогов.

Выбор текущего рабочего каталога является одной из общих настроек приложения (главное окно).

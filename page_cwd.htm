КАТАЛОГ Съёмок

Для управления проектами исследования пещер, включающими несколько съёмок, желательно использовать определенный рабочий каталог (папку) для каждого проекта пещеры.
Каждый рабочий кататлог является подкаталогом, который создается автоматически. Имя текущего рабочего каталога должно начинаться с "TopoDroid". Если вы укажете имя, которое не начинается с "TopoDroid", то к нему будет добавлен префикс.
Папки проекта создаются внутри базовой папки TopoDroid в "Documents/TDX".

Символы для абрисов и файлы, связанные с приборами ( базы данных устройств, прошивки, дампы памяти, калибровки), находятся только в системном каталоге "files" во внешнем хранилище. Другие рабочие каталоги содержат только рабочие файлы базы данных съёмки и файлы топосъёмки.

Для управления проектами топосъемки пещер, включающими несколько отдельных съемок, желательно использовать определенный рабочий каталог (папку) для каждого проекта пещеры.
Название любого рабочего каталога должно начинаться с "TopoDroid" (не чувствительно к регистру).
Рабочие каталоги создаются внутри базовой папки TopoDroid: "Documents/TDX".

Файлы условных обозначений и файлы, относящиеся к устройствам. (database, firmware, memory dumps, calibrations) хранятся только в каталоге системных файлов приложения (в памяти телефона). Рабочие папки содержат только базу данных съёмок и папку съёмок.

Выбор текущего рабочего каталога является одной из общих настроек приложения (главное окно).

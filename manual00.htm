TopoDroid - это Android-приложение для топосъемки пещер. TopoDroid собирает данные топосъёмки, помогает вам организовать их и построить карту нитки хода прямо на месте. Вы можете дополнительно рисовать стены и другие объекты вокруг нитки хода во время съёмки, тем самым снижая вероятность ошибки в конечном результате, и можете обнаружить ошибки в данных, пока вы все еще в пещере. TopoDroid имеет множество функций, которые помогут вам получить максимум от вашей работы в пещере. Таким образом, примерная карта будет почти готова уже по окончании рабочего выхода.

TopoDroid делает процесс топосъемки пещеры более эффективным. Он предназначен для работы только с одной съёмкой. Он не предназначен для управления сложными проектами, которые включают множество съемок. Это сделано потому, что цель TopoDroid заключается в том, чтобы помочь спелеологу выполнять топосъемку с большей точностью, а также сделать процесс топосъемки более приятным, а не производить построение окончательной карты пещеры. Для этой цели уже существуют хорошие программы для PC.

В TopoDroid существует пять разных "уровней функционала". На самомо базовом уровне есть все необходимые функции чтобы получать данные с Дисто, управлять номерами пикетов и рисовать простые абрисы. Более высокие и богатые функциями уровни могут быть использованы для болле детальной проработки карт и подготовки к обработке на ПК.

TopoDroid предназначен для работы с одной единственной съемкой, а не для создания карт крупных пещерных систем, состоящих из многих съёмок. Однако програма включает в себя Cave Project Manager, который может быть использован для соединения отдельных съёмок в проект пешеры или карстового района, а также 3D Viewer, с помощью которого вы можете просматривать 3D съёмки и проекты пещер.

TopoDroid специально разработан для работы с DistoX, SAP5, и BRIC4. Однако при этом данные топосъемки могут быть введены и вручную.

Внимание Устройства Android (а также активный стилус) могут влиять на измерения сделанные с помощью DistoX/BRICK, если находятся слишком близко.

Робот Android воспроизводится или модифицируется на основе работы, созданной и предоставленной Google, и используется в соответствии с условиями, описанными в лицензии Creative Commons 3.0 Attribution.

Системный каталог TopoDroid

Основной каталог программы TopoDroid находится в системной папке "Android/data/com.topodroid.TDX/files", в которой находятся подкаталоги для инструментов рисовния, "point", "line" и "area", а также файл базы данных устройств "device10.sqlite", содержащий данные калибровки, экспортипрванные данные калибровки (каталог "ccsv"), прошивки DistoX2 (каталог "bin"), и дампы памяти DistoX (каталог "dump").

Каталог и его содержимое удаляются при деинсталяции программы. Но их можно сохранить если включить опцию "keep the app data"


Каталог съёмок TopoDroid

Данные съёмок и абрисы сохраняются в поддиректорию "Documents/TDX", в основной памяти телефона. Съемки хранятся в каталоге "TopoDroid". Каждая папка съёмки содержит файл базы данных "distox14.sqlite", каталоги "zip", "tmp" для временных файлов, а также папки для каждой новой съёмки.

Каталог и его содержимое не удаляются при деинсталяции программы.

Данные съёмки

Данные съёмок и абрисы сохраняются в поддиректорию "TopoDroid" в рабочем каталоге TDX. Для каждой съемки создается подкаталог в рабочей папке "TopoDroid".
Папка "TopoDroid" содержит файлы базы данных "distox14.sqlite", а также каталоги отдельных съемок, а также папку проектов пещер "thconfig"

Предоставление прав доступа

На Android-11 и выше, TopoDroid может не иметь возможности создать базовую папку "TDX", или какие-то вложенные папки и файлы в ней, потому что у него недостаточно прав для доступа к внешнему хранилищу.
В этом случае вы можете предоставить разрешения, используя "adb" (Android Debug Bridge).
  1. скачайте "adb" на ваш PC
  2. в Android, включите "Developer options" (заходим в настройки "О телефоне", тапаем семь раз "Номер сборки")
  3. включите "USB debugging" в "Developer options"
  4. на PC запустите adb server. Введите команду "adb start-server"
  5. подсоедините Android к PC с помощью USB кабеля. Возможно, вам придется разрешить adb-соединение с PC, в вашем Android
  6. в терминале на PC введите слкдующие три команды:
    • adb shell appops set com.topodroid.TDX READ_EXTERNAL_STORAGE allow
    • adb shell appops set com.topodroid.TDX WRITE_EXTERNAL_STORAGE allow
    • adb shell appops set com.topodroid.TDX MANAGE_EXTERNAL_STORAGE allow

Данные съёмок и абрисов не удаляются в случае если программа была деинсталирована.

Импорт съёмок старых верисий

TopoDroidX (v.6) можно установить вместе со старым v. 5.1.40потому что они используют разные рабочие директории.
Съймки, сделанные в старых версиях программы могут быть импортированы в TopoDroidX v.6 в виде zip архивов, созданных старыми версиями программы.
Если у вас нету zip архивов, вы можете создать их с помощью следующих скриптов zip_script.

N.B. TopoDroid v.5.1.40 не может импортировать zip архивы, созданнык в v.6 из-за изменений в структуре базы данных.

Используемые сенсоры

TopoDroid может использовать следующие функции телефона: "bluetooth", "камера", "микрофон", "GPS", "мультитач". TopoDroid использует интернет соединение исключительно для получения и установки руководства пользователя. Функции TopoDroid, которым необходимо оборудование, отсутствующее в вашем аппарате, будут автоматически отключены.
Интернет-соединение используется только для установки переводов руководства пользователя.

Ошибки и сбои

Из-за ограниченности ресурсов свежие версии программы не могут быть полностью протестированы. Рекомендуется устанавливать обновления TopoDroid при помощи APK-менеджера, который сохраняет APK ранее установленных версий. Таким способом вы можете безопасно вернуться к более стабильной версии, в случае, если у вас возникнут проблемы с последними APK. Не забудьте сообщить о своих проблемах, чтобы они могли быть приняты во внимание и решены в следующей версии. Отчеты об ошибках будут оперативно рассмотрены и учтены в следующих обновлениях.
  • Если приложение "упало" и вы получили отчет об ошибке, пожалуйста перешлите этот отчет через систему Android.
  • Если вы нашли ошибку или приложение не ведет себя так, как вы ожидаете, напишите мне по электронной почте.
    • Не забудте указать версию приложения и версию ос Android.
  • Ошибки при выборе опции из списка могут быть связаны с неполным переводом списка опций. Перезапустите приложение переключив язык на Английский; если ошибка пропала, свяжитесь с человеком, который делал перевод.
  • Если у Вас есть вопрос или запрос на новую функцию, отправьте мне письмо по электронной почте. Не забудьте включить номер версии приложения.

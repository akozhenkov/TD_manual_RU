List and Grid tool picker

This dialog is used to select the current drawing tool.
It is opened tapping the Tool button in the Sketch window (in "drawing" mode), provided the "tool picker" setting is set accordingly.

The three tabs at the top, Point, Line, Area switch among the three tool types.

The dialog displays the tools of the selected type

  • either as a list, with a selection checkbox, tool icon and name
  • or a as a grid with the tool icons
To select a tool
  • check its box in the list, and close the dialog (Back key)
  • tap its icon in the grid, and close the dialog

A shortcut for these is a double-tap on the check-box or the icon.

Slider set icon orientation (orientable point tools).

+/-: change icon size (XS, S, M, L, XL).

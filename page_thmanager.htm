ThManager

Приложение ThManager управляет проектами Therion. Проекты храняться в конфигурационных файлах Therion в директории "thconfig". С помощью этого приложения можно:
  • создать проект Therion
  • открыть существующий проект
  • удалить проект
  • длбавить файлы данных ".th" к проекту
  • удалить файлы данных из проекта
  • соединять нитки хода в графическом режиме либо при помощи команды "equate"
  • удалять соединения "equates"
  • отображать 3D структуру пещеры (при помощи Cave3D)
Основные кнопки:
  • Создать проект
  • изменить настройки
  • помощь
  • выход

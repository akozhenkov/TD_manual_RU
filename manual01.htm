Уровни функционала

Интерфейс TopoDroid имеет уровни функционала, различающихся по количеству доступных функций.
  • В базовом уровне [B] рисование ограничено линиями, без каких-либо действий по редактированию.
  • Уровень нормальный [N] содержит все функции, необходимые для детальной съёмки пещеры.
  • Уровень продвинутый [A] содержит все функции.
  • Уровень эксперементальный [E] предназначет для развития. В нем дополнительно доступны экспериментальные возможности. Эти функции лиш частично отлажены и могут вызывать ошибки. Пожалуйста, сообщайте пожалуйста любую проблему с новыми функциями.

По умолчанию в TopoDroid включен только нормальный функционал, но вы можете переключить уровень, в соответствии с вашими потребностями не выходя из программы. В данном руководстве описывается весь функционал TopoDroid. Функции на уровне выше "базового" помечены литерой уровня [N], [A] или [E].

Окна

Из главного окна TopoDroid вы можете открыть другие функции приложения. Основными функциями являются:
  • Топосъёмка управление данными (съёмка и окно списка пикетов).
  • Зарисовка абрисов (окно рисования) является подчиненным действием топосъёмки.
  • DistoX управление (окно управления приборами)
  • Калибровка DistoX (окна калибровки и данных-калибровки) запускается из окна устройств.


Окна Topodroid имеют прокручиваемую панель кнопок в верхней части для наиболее частых действий, и кнопку меню в верхнем правом углу. Диалоги обычно имеют кнопку подтверждения(OK или Сохр.). Если вы хотите закрыть диалоговое окно или отменить все, что вы сделали, нажмите на Android кнопку НАЗАД. Если есть обе кнопки OK и Сохранить первая сохраняет входные данные и закрывает диалоговое окно, в то время как вторая сохраняет входные данные и оставляет диалог открытым для ввода новых данных. Информационные диалоги не имеют никаких кнопок.

В любом окне TopoDroid аппаратная кнопка МЕНЮ открывает диалог Настроек

Размер текста и кнопок

Параметр "Размер текста" определяет размерность текста в списках, в частности, перечень съёмок, пикетов и калибровочных данных.
Есть два варианта для размера кнопок: "маленькие" и "большие". Первый подходит для сотовых телефонов (диагональ экрана 5'), и используется умолчанию. Второй подходит для планшетов (диагональ экрана 7').

Инструменты для рисования (палитра)

TopoDroid поставляется с большим количеством встроенных инструментов для рисования. По умолчанию включены только самые основные инструменты. Вы можете включить больше инструментов по мере необходимости, с помощью меню "рисование". Таким способом вы можете понемногу привыкнуть к "иконографии" TopoDroid.

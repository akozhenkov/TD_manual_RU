Уровни функционала

Интерфейс TopoDroid имеет уровни функционала, различающиеся по количеству доступных функций:
  • В базовом уровне [B] рисование ограничено линиями, без каких-либо действий по редактированию.
  • Уровень нормальный [N] поддерживает все функции, необходимые для детальной съёмки пещеры.
  • Уровень продвинутый [A] поддерживает все функции.
  • Уровень экспертный [E] предназначен для разработки. В нем доступны экспериментальные возможности. Эти функции лишь частично отлажены и могут вызывать ошибки. Пожалуйста, сообщайте о любой проблеме с новыми функциями.

По умолчанию в TopoDroid включен только нормальный функционал, но вы можете переключить уровень в соответствии с вашими потребностями, не выходя из программы. В данном руководстве описывается весь функционал TopoDroid. Функции на уровне выше "базового" помечены литерой уровня [N], [A] или [E]. Опции диалогов настройки также зависят от выбранного уровня фцнкционала.

Окна

Из главного окна TopoDroid вы можете открыть другие функции приложения. Основными функциями являются:
  • Топосъёмка: и управление данными (съёмка и окно списка пикетов).
  • Зарисовка: абрисов (окно рисования) - является подчиненным действием топосъёмки.
  • DistoX: управление (окно управления приборами)
  • Калибровка: DistoX (окна калибровки и данных-калибровки) - запускается из окна Устройств.


Окна Topodroid имеют в верхней части прокручиваемую панель кнопок для наиболее частых действий, и кнопку меню в верхнем правом углу. В диалогах обычно есть кнопка подтверждения (OK или Сохр.). Также в них может присутствовать кнопка "Назад" (или "Отмена"). Если вы хотите закрыть диалоговое окно или отменить все, что вы сделали, нажмите на кнопку НАЗАД. Если есть обе кнопки: OK и Сохранить первая сохраняет входные данные и закрывает диалоговое окно, в то время как вторая сохраняет входные данные и оставляет диалог открытым для ввода новых данных. Информационные диалоги не имеют никаких кнопок.

В любом окне TopoDroid аппаратная кнопка МЕНЮ открывает страницу Справки.

Размер текста и кнопок

Параметр "Размер текста" определяет размерность текста в списках, в частности, списке съёмок, пикетов и калибровочных данных.
Есть пять вариантов размера кнопок: "маленький", "нормальный", "средний", "большой" и "огромный". "Нормальный" подходит для телефонов (диагональ экрана 5') и используется по умолчанию. "Большой" подходит для планшетов (диагональ экрана 7').

Инструменты для рисования (палитра)

TopoDroid поставляется с большим количеством встроенных инструментов для рисования. По умолчанию включены только самые основные инструменты. По мере необходимости вы можете включить больше инструментов с помощью кнопки "Палитра" в главном окне. Таким способом вы можете понемногу привыкнуть к "иконографии" TopoDroid.

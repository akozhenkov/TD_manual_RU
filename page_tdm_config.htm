TopoDroid Manager - Окно Проекта

Окно проекта пещеры открывается при нажатии имени проекта в списке менеджера проектов.


It displays the list of surveys that comprise the project. The list is empty if the project has no surveys.
Each survey can be selected for further actions, by tapping its checkbox.

Меню
Закрыть окно
Экспортировать проект пещеры (Therion или Survex)
Удалить проект пещеры
Помощь

Кнопки

  • Добавить съёмки
  • Удалить выбранные съёмки
  • Показать выбранные съёмки в плане
  • Показать список геопривязанных пикетов
  • Показать список соединений
  • Показать съёмку в Cave3D

    • Кнопка вставить отображает список съемок в базе данных TopoDroid. Съемки добавляются в проект, устанавливая их флажки и подтверждая нажатием кнопку ОК.

    Project 3D view
    Проект отображается в3D view, начиная с первой съемки.
    Следующие съемки отображаются в 3D только в том случае, если они (прямо или косвенно) связаны с первой съемкой.

    Если несколько съемок имеют географическую привязку, все они могут отображаться в 3D вместе с подключенные опросы.

    Cъемки в плане 2D
    В окне отображаются съемки в плане с ниткой хода и названиями пикетов. Каждая съемка имеет свой цвет. Через некоторое время цвета будут повторятся, поэтому вы должны отображать только две или три съемка за раз.
    Сопряженные пикеты связаны пунктирными красными линиями, и выделены круглым фоном. Размер имен пикетов такой же, как и в окне абриса.

    Окно можно передмещать и масштабировать.
    Съемку можно захватить (нажмите на пикет: он будет обведен) и перетащить относительно других съемок. Таким образом вы можете совмещать пикеты разных съемок, которые вы хотите отождествить. Когда они пересекаются, обведите их обоих, нажав на точку: затем вы можете создать сопряжение между ними с помощью кнопки «Добавить сопряжение».

    Есть две кнопки

    • Добавить сопряжение: открывает диалоговое окно для добавления вручную сопряжения между съемками проекта пещеры.
    • Список сопряжений между съемками проекта пещеры

    Сопряжения
    Вы можете удалить сопряжение, коснувшись его записи в диалоговом окне, в котором перечислены сопряжения проекта пещеры.

    ОКНО ПРОСМОТРА СЪЁМОК 2D
    Съемки отображается на плане в виде нитки хода с номерами пикетов.
    Каждая съёмка отображается своим цветом. Через некоторое время цвета будут повторятся, поэтому вам лучше отображать только две или три съёмки одновременно. Соединения показаны пунктирными красными линиями, а соединенные пикеты показаны в круге. Размер подписей пикетов соответствует размеру, указаномму в настройках окна рисования топодройд.
    Карту можно передвигать и увеличивать. Съемку можно переместить (нажать на пикет: он будет показан в круге и перетащить ее), в то время как другие съемки не перемещаются. Таким образом, вы можете совмещать пикеты из разных съемок, которые вы хотите соединить. Когда номера пикетов перекрывают друг друга обведите их, коснувшись точки: затем вы можете вставить соединение между ними с помощью кнопки "Добавить соединеиние".

    КНОПКИ

    • Добавить соединение: открывает диалоговое окно, которое отображает объединяемые пикеты. Чтобы вручную добавить соединение между съемками проекта пещеры, нажмите кнопку ОК.
    • Список соединений между съемками проекта пещеры

    Вы можете удалить соединение, нажав на него в диалоговом окне, в котором перечислены соединения проекта пещеры.

DistoX ВЫБОР ПРИБОРА

TopoDroid поддерживает работу со следующими приборами:
  • DistoX (первая версия)
  • DistoX2 (вторая версия)
  • SAP5 (Shetland Attack Pony), только модель со встроенным Bluetooth [T]
Все приборы DistoX должны быть сопряжены с устройством Android.

Обычно TopoDroid работае только с одним прибором. Передача данных может осуществляться постоянно (постоянный режим) или порциями (режим по требованию). Прибор SAP5 работает только в режиме постоянной передачи данных.

TopoDroid может одновременно работать с несколькими DistoX, но обмен данными происходит только с одним из DistoX в один момент времени.
Pежим обмена данными устанвлен на мульти. В этом случае при длинном нажатии на кнопку скачать появться диалог, в котором можно выбрать с какого DistoX получать данные измерений.

DistoX ВЫБОР ПРИБОРА

TopoDroid может одновременно работать с несколькими DistoX, но обмен данными происходит только с одним из DistoX в один момент времени.
Все приборы DistoX должны быть сопряжены с устройством Android, а режим обмена данными устанвлен на мульти. В этом случае при длинном нажатии на кнопку скачать появться диалог, в котором можно выбрать с какого DistoX получать данные измерений.

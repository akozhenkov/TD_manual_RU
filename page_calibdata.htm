Диалоговое окно данных калибровки

Нажатие на один из пунктов в списке калибровок открывает дивлоговое окно работы с данными калибровки.

Через этот диалог номер группы для каждого измерения может быть назначен или изменен: используйте целое положительное число для данных, которые принимают участие в расчете калибровочных коэфицентов, 0 (ноль) для данных, которые не принимают участие в расчете, и -1 (минус один) для "удаленных" данных. Удаленные данные обычно не показываються в списке калибровочных замеров и игнорируються при создании групп замеров.

Этот диалог содержит три кнопки:

  • если опция Перегруппировать включена, то при сохранении изменений, TopoDroid переформирует группы данных начиная с выбранной записи с использованием текущей политики группировки.
  • кнопка Сохранить сохраняет номера групп и закрывает диалог.
  • кнопка Удалить "удаляет" замер и закрывает диалог. Удаленный замер, по умолчанию не отображается в списке. Но вы можете включить отображение удаленных замеров в меню настроек отображения Показать. Удаленные измерения будут показаны на сером фоне. Если вы нажмете на удаленный замер то вам будет предложено восстановить его.

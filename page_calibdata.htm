Диалоговое окно данных калибровки

Нажатие на один из пунктов в списке калибровок открывает диалоговое окно работы с данными калибровки.

Через этот диалог может быть назначен или изменен номер группы для каждого измерения: используйте целое положительное число для данных, которые принимают участие в расчете калибровочных коэффициентов, 0 (ноль) для данных, которые не принимают участие в расчете, и -1 (минус один) для "удаленных" данных. Удаленные данные обычно не показываются в списке калибровочных замеров и игнорируются при создании групп замеров.

Этот диалог содержит три кнопки, слева направо:

  • Перегруппировать: если опция включена, то при сохранении изменений, TopoDroid переформирует группы данных, начиная с выбранной записи, с использованием текущей политики группировки.
  • Сохранить: сохраняет номера групп и закрывает диалог.
  • Удалить: "удаляет" замер и закрывает диалог. Удаленный замер, по умолчанию не отображается в списке. Но вы можете включить отображение удаленных замеров в меню "настроек отображения": Показать. Удаленные измерения будут показаны на сером фоне. Если вы нажмете на удаленный замер, то вам будет предложено восстановить его.

Location GPS

This dialog is opened tapping the GPS button of the Locations dialog.

To use the GPS the Android Location must be enabled.

This dialog has

  • station name
  • comment
  • GPS starts/stops the location
  • View the point in a location app
  • Save the point
While searching for satellites the "GPS" button becomes red (or green) displays the number of satellites in sight. You should wait till the button becomes green and the coordinates are displayed underneath. Then stop the GPS location tapping the button again.

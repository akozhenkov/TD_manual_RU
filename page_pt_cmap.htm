PocketTopo color map

This dialog is opened selection the PocketTopo color-map setting (in the Import/Export section).

It displays a table with three columns.

  • PocketTopo color
  • associated line (Therion name)
  • associated point symbol (Therion name)

The color-map is used in the import of ".top" files, to map PocketTopo colors to TopoDroid drawing items.

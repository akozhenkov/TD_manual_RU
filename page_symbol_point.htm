ИНСТРУМЕНТЫ "ТОЧЕК"

Инструменты "точки" заданы в текстовых файлах.

В программу встроены два инструмента "точек": "пользовательский" и "подпись"(текст). Остальные загружаются из подкаталога TopoDroid symbol/point. Этот каталог заполняется инструментами "точек", упакованными в дистрибутив APK. Вы можете удалить инструменты, которые вам не нужны, редактировать те, которые вы хотите изменить, и добавлять новые, не вошедшие в APK.

Файл инструмента "точка" содержит данные, определяющие свойства инструмента рисования:

  • "symbol point" [обязательный]
  • "name NAME", где NAME название точки по умолчанию [обязательный]
  • переводы названия, например "name-it ..." (Итальянский) [не обязательный]
  • "th_name NAME", где NAME это название в Therion, и должно совпадать с именем файла [обязательный]
  • "color rgb" rgb цвет в шестнадцатиричном формате [обязательный]
  • "orientation FLAG" не ориентируемо, если нет флага, ориентируемо если FLAG '1' или 'yes' [не обязательный]
  • "has_text FLAG" нет подписи, если нет флага, есть подпись если FLAG '1' или 'yes' [не обязательный]
  • "csurvey LAYER TYPE CATEGORY" категория инструмента csurvey
  • "level" view_flag [optional, 1 if missing]
  • "path" начало кода, определяющего символ точки. Символ задается набором следующих команд:
    • "moveTo X Y"
    • "lineTo X Y"
    • "cubicTo X1 Y1 X2 Y2 X Y"
    • "addCircle X Y R": (X,Y) координата центра окружности, R радиус окружности (добавить окружность)
    • "arcTo X0 Y0 X1 Y1 FROM_ANGLE TO_ANGLE": (X0,Y0 - X1,Y1) ограничивающий квадрат, угол в градусах, 0 вдоль оси X, направление против часовой стрелки.
  • "endpath" закрывает секцию кода точки
  • "endsymbol"
Ориентация
Если атрибут "ориентация" установлен, символ точки может быть повернут.

Слой
Атрибут "level" считывает значение флага для того чтобы отображать точку только на определенном слое.
Флаг это сумма чисел 1 (базовый), 2 (пол), 4 (отложения), 8 (потолок) и 16 (артефакты).

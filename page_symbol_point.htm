ИНСТРУМЕНТ ТОЧЕК

Инструменты "области" заданы в текстовых файлах.

Два инструмента точек, "пользовательский" и "подпись"(текст), встроены в программу. Остальные загружаются из подкаталога TopoDroid symbol/point. Этот каталог заполняется инструментами точек, упакованных в дистрибутив APK. Вы можете удалить инструменты, которые вам не нужны, редактировать те, которые вы хотите изменить, и добавлять новые, не вошедшие в APK.

Файл инструмента точка содержит данные, определяющие свойства инструмента рисования.

  • "symbol point" [обязательный]
  • "name NAME", где NAME название точкм по умолчанию [обязательный]
  • переводы названия, т.е. "name-es ..." [не обязалельный]
  • "th_name NAME", где NAME это название в Therion, и должно совпадать с именем файла [обязалельный]
  • "color rgb" (rgb цвет в шестнадцатиричном формате) [обязалельный]
  • "close-horizontal" [optional, false if missing???ошибка]
  • "orientation FLAG" [не обязательный, не ориентируемо если нет флага,ориентируемо если FLAG '1' или 'yes']
  • "has_text FLAG" [не обязательный, нет подписи если нет флага, есть подпись если FLAG '1' или 'yes']
  • "csurvey LAYER TYPE CATEGORY" категория инструмента csurvey
  • "path" begins the point symbol path. This is specified with the following commands
    • "moveTo X Y"
    • "lineTo X Y"
    • "cubicTo X1 Y1 X2 Y2 X Y"
    • "addCircle X Y R"
    • "arcTo X0 Y0 X1 Y1 FROM_ANGLE TO_ANGLE" (direction is CCW)
  • "endpath" marks the end of the tool path
  • "endsymbol"
Ориентация
Если атрибут "ориентация" установлен, символ точки может быть повернут.

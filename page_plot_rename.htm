Диалог переименования абриса

При помощи этого диалога вы можете изменять название абриса. Он открывается при нажатии на пункт меню Переименовать в окне Рисования.

Впишите новое назввание абриса в поле для ввода и нажмите OK для подтверждения.

Для отмены нужно закрыть диалог Android кнопкой НАЗАД.
